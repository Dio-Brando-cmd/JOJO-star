# 帷幕之地 2D玩法扩展设计

> 设计日期：2026-07-23 | 状态：设计稿

---

## 扩展一：血月事件系统

### 触发机制
- 每轮帷幕结算后翻事件牌堆顶一张
- 牌堆由3张血月+4张空牌组成（共7张）
- 同局血月事件不重复触发
- 第三张血月后不再补充牌堆

### 事件一：猩红帷幕
- 所有技能冷却减半（向上取整）
- 每次使用技能失去1点灵焰值
- 蚀者伤害+1，守幕者治疗+1

### 事件二：破碎镜界
- 所有角色强制停留在当前格，无法移动
- 信息技能范围扩大2格
- 存活玩家可进行一次20秒公开短对话

### 事件三：魂印逆流
- 所有魂印触发条件反转（夜晚↔白昼）
- 每个玩家随机获得一个临时魂印（不与已有重复）
- 若临时魂印与自身魂印构成羁绊对，可立即触发一次

## 扩展二：灵植道具系统

### 采集
- 白昼新增"寻灵"子阶段（投票前）
- 6个采集点（从12个预设坐标随机选出）
- 按行动力降序行动，消耗1行动力移动+采集

### 灵植类型
| 灵植 | 产出点 | 制作 |
|------|--------|------|
| 银叶草 | 4个 | 护身灵符 |
| 暗蕊花 | 1个 | 真视灵符 |
| 虚空苔 | 1个 | 扰乱灵符 |

### 三种灵符
| 灵符 | 效果 |
|------|------|
| 护身灵符 | 免疫一次蚀者直接技能（不能免疫投票） |
| 真视灵符 | 选择一名玩家，获知阵营；第三方额外获知碎片数 |
| 扰乱灵符 | 目标本轮技能随机偏移至相邻合法目标 |

### 规则
- 每人最多持有2张灵符
- 制作后下一轮白昼可用
- 不可交易、不可丢弃
- 同一采集点每轮只能采集一次（先到先得）

## 扩展三：冥途商人（第三方阵营）

### 属性
- 独立第三方，胜利条件与其他阵营互斥
- 初始灵焰碎片：0 | 行动力：2
- 侦查技能对其返回"守幕者阵营"

### 交易（白昼投票前）
- 消耗1行动力，选择一名玩家发起
- 提供一条可验证真实信息，目标支付碎片（最少1枚，不超过目标持有量半数）
- 三种信息类型：两玩家阵营是否相同 / 一死者身份 / 剩余血月牌数

### 夜隐（夜晚）
- 选择一名玩家获得"夜隐"——免疫蚀者技能，但守幕者探查结果被篡改为"蚀者阵营"
- 不可对自己使用

### 胜利条件
| 人数 | 碎片阈值 |
|------|---------|
| 8人 | 15枚 |
| 10人 | 20枚 |
| 12人 | 25枚 |

## 扩展四：魂印羁绊系统

### 分组
15魂印分5组"羁绊三环"：(1,6,11)(2,7,12)(3,8,13)(4,9,14)(5,10,15)

### 触发条件
1. 投票目标一致
2. 曼哈顿距离≤3
3. 至少一人持有灵焰碎片

### 五组效果
| 组 | 效果 |
|----|------|
| 共鸣之环 (1,6,11) | 两人各+1碎片，可额外私聊30字 |
| 守护之环 (2,7,12) | 消耗1碎片抵消致命伤害（每轮限1次） |
| 洞察之环 (3,8,13) | 获知对方灵符种类数量，侦查范围+1 |
| 暗影之环 (4,9,14) | 位置标记可交换，蚀者技能50%偏移 |
| 涅槃之环 (5,10,15) | 消耗3碎片使被投票淘汰者存活（全局1次） |
