# 帷幕之地 · 3D玩法全景设计

> 基于世界观"灵焰纪元"，覆盖五种3D游戏模式的全景设计文档
> 设计日期：2026-07-23

---

# 一、总览

| # | 模式 | 类型 | 人数 | 单局时长 | 核心体验 |
|---|------|------|------|----------|----------|
| 1 | **帷幕追猎** | 非对称PvP | 2v6 | 15-20min | 类似Dead by Daylight的追逃紧张感 + 灵焰争夺 |
| 2 | **守幕者之夜** | PvE合作 | 5人 | 25-35min | 波次防守+Boss战，资源管理与职业协作 |
| 3 | **暗幕推理** | 社交推理+潜入 | 10人 | 30-45min | Among Us式的昼夜循环 + 五种不同的灵焰视野 |
| 4 | **裂隙竞技场** | 对称PvP | 3v3 | 10-15min | 抢占裂隙能量节点，阵营混编对战 |
| 5 | **帷幕之域** | 开放世界 | 1-4人 | 自由 | 六区域探索、生存建造、灵格养成 |

# 二、帷幕追猎：非对称PvP（2v6）

## 核心循环

追猎方（蚀者+冥僧人）需在120秒夜晚中吞噬/堕化4名守幕者。守幕方可存活至黎明、完成放逐仪式或直接击杀追猎方。

## 追猎方能力

| 蚀者技能 | 冷却 | 效果 |
|----------|------|------|
| 噬灵突袭 | 8s | 冲刺8m，命中吞噬目标40%灵焰，三次耗尽=击杀 |
| 灵痕追踪 | 被动 | 30s内所有守幕者足迹以暗红色光带可见 |
| 裂隙穿越 | 25s | 3s后传送至30m内任意裂隙，1.5s现身动画 |
| 蚀痕爆发 | 60s | 半径15m冲击波，范围蚀痕标记+25%后续伤害 |

| 冥僧人技能 | 冷却 | 效果 |
|------------|------|------|
| 灵焰伪装 | 20s | 30s内以守幕者灵焰频率出现，灵织者无法辨识 |
| 堕化低语 | 45s | 目标进入"被低语"状态30s，冥僧人可看到其视野 |
| 裂隙引导 | 60s | 在指定位置开启裂隙30s，附近守幕者移速-20% |
| 虚空契约 | 全局1次 | 标记一名守幕者，其灵焰将在90s后被虚空吞噬 |

## 守幕方关键能力

| 职业 | 3D核心能力 |
|------|-----------|
| 帷幕学者 | 精确察灵识破冥僧人伪装(10m内)；溯灵回溯回放30s灵焰残影；幕谕预警Boss技能方向 |
| 草药学者 | 采集灵植(每局12株刷新点)；蚀灭药剂(远程投掷范围伤害)；愈灵符(远程救起倒地队友)；灵质烟雾(遮蔽蚀者灵痕追踪10s) |
| 愈灵师 | 灵焰修复(单体治疗)；净化触觉(反制冥僧人堕化)；灵焰共鸣(15m群体治疗光环)；濒死救援(从噬灵中拉回队友) |
| 帷幕守卫 | 守护誓言(绑定一人分担伤害)；灵质门锁(锁门使蚀者必须攻击破门)；筑垒结界(8m无敌区域15s)；舍身守护(替绑定者承受致命伤害) |
| 灵痕追猎者 | 灵焰猎枪(远程70m射击，击中蚀者使其3s无法移动)；噬灭短铳(被近身时击退+短眩晕)；猎杀姿态(每击杀一个蚀者增伤10%，可叠加) |
| 灵织者 | 信息网络核心：五个子类型(老兵/行路/守夜/锻甲/织幕)各自不同的灵焰感知范围和低语解读能力 |

## 地图交互

- **门锁四级**：无锁→普通锁(3s破门)→灵质加固(8s破门)→帷幕加固(无法进入)
- **地标功能**：枯树广场(放逐仪式点)、水井(藏匿+秘密通道)、铁匠铺(锻造灵符)、观测塔(全图侦查)、墓地(灵焰残响信息)

## 胜负与隐藏结局

- 追猎方胜：吞噬/堕化≥4人
- 守幕方胜：存活至黎明 / 3人20s仪式放逐 / 直接击杀追猎方
- **深渊永夜**(隐藏)：若120s内守幕方全员被吞噬且冥僧人借贷归零→千面之影部分降临，影响后续5局
- **余烬之路**(隐藏)：若守幕方仪式放逐成功且0死亡→该局所有玩家获得永久灵格加成

# 三、守幕者之夜：PvE合作（5人）

## 核心循环

五名守幕者保护聚落核心"帷幕圣坛"，在五波蚀者潮中存活至黎明。管理共享灵焰池(1500点)、制作灵符、修复裂隙。

## 波次设计

| 波次 | 蚀者类型 | 特殊机制 |
|------|---------|----------|
| 1 | 8-10普通蚀者+1小裂隙 | 教学波，守幕者可自由尝试技能 |
| 2 | 蚀痕射手(远程)+蚀痕卫士(高防)+中型裂隙 | 引入多敌种协同 |
| 3 | 15杂兵+蚀痕尖叫者+栅栏塌陷 | 数量碾压+紧急事件 |
| 4 | 冥僧人学徒精英×3 | 蚀痕链接(共享生命)+灵焰汲取+裂隙共振 |
| 5 | 冥僧人首领Boss+裂隙不断涌出 | 最终决战 |

## 三种Boss战

### Boss 1：蚀痕巨兽（3000HP）
- 每损失30%HP分裂为两个小型巨兽，需同步击杀
- 站在裂隙上每秒回血5%，必须先用愈灵符封住裂隙
- 胸腔核心弱点头部并非弱点，需在巨兽扑击后暴露的3秒窗口攻击

### Boss 2：厄尔苏拉斯——冥僧人首领（4500HP）
- **学者审判**：周期性释放全屏"审判"判定——所有正在移动的玩家受到持续灵焰灼烧。正确应对是**停下脚步**（反直觉设计）
- **灵焰反转**：将守幕方的治疗变为伤害，伤害变为治疗——持续15s，团队需要识别并调整行为
- **契约背弃**：随机选择一名玩家，给予30%增伤但需"献祭"另一名队友(攻击队友)→选择拒绝则自身受到60s的80%减疗debuff→道德困境测试

### Boss 3：千面之影化身（6000HP）——隐藏Boss
- 仅在收集齐10个以上叙事碎片时触发
- **镜像复制**：复制当前所有守幕者的职业能力并以120%强度释放
- **全队自述仪式**：全队进入灵质空间，每个玩家必须用一段话描述自己角色的核心价值(如"我守护他人，是因为我知道被放弃的滋味")。**全队正确完成→千面之影被放逐；有一人错误→直接团灭**——这是一个考验玩家是否真正理解自己角色背景的终极机制
- **千人千面**：每个玩家看到的千面之影形态不同(程序根据该玩家的游戏历史生成)——有人看到逝去的亲人，有人看到曾经的自己

## 四结局

| 结局 | 条件 | 演出 |
|------|------|------|
| 帷幕黎明 | 全队存活+高资源+击败千面之影 | 日舟升起，帷幕发出淡金光芒，解锁金色灵格印记 |
| 余烬之守 | 有人永久出局但胜利 | 幸存者围在死者灵焰消散处，获得"余烬"称号 |
| 蚀痕折中 | 存活但未触发隐藏Boss | 平淡收场，灵脉碑读数堪忧 |
| 千面终临 | Boss未被击败 | 千面之影的碎片进入帷幕，影响**接下来5局竞技模式** |

# 四、暗幕推理：社交推理+3D潜入（10人）

## 角色分配
2蚀者 + 2灵织者 + 1帷幕学者 + 1帷幕守卫 + 4普通人

## 核心系统：五种灵焰视野（同一世界不同渲染）

| 视野类型 | 使用者 | 画面特征 | 额外信息 |
|---------|--------|---------|----------|
| 常规视野 | 普通人 | 物理写实、手电筒照明 | 仅肉眼可见线索 |
| 灵焰视野 | 灵织者 | 青蓝滤镜、发光轮廓 | 15s灵痕尾迹(含颜色)、尸体残响回放 |
| 察灵视野 | 帷幕学者 | 琥珀色调 | 蚀刻印(10m内)、地面蚀斑、裂隙噪点 |
| 追迹视野 | 帷幕守卫 | 深蓝色调 | 蚀痕热力图(30s)、灵缚标记追踪 |
| 噬灵视野 | 蚀者 | 偏暗、目标高亮 | 灵脉节点标记、15m外孤立目标金色轮廓 |

## 昼夜循环

**白天(8min)：** 3D聚落自由探索 → 采集灵植/加固房屋/调查尸体残响/阅读瞭望塔日志 → 敲钟发起会议

**夜晚(3min)：** 蚀者暗中行动(噬灵/堕化/灵脉破坏/尸体搬运) → 帷幕守卫夜巡追踪 → 普通人受限居民区

## 会议与放逐演出

- 投票超半数→放逐至帷幕裂隙
- 放逐蚀者：裂隙炸裂紫光 + 灵脉腐蚀度-20% + 全屏净化特效
- 放逐好人：裂隙哀嚎 + 灵脉腐蚀度+15% + 画面龟裂特效
- **身份不直接公布**——玩家通过演出效果自行判断

# 五、裂隙竞技场：对称PvP（3v3）

## 核心玩法

双方各选3人，每队必须包含至少1名蚀者阵营和至少1名守幕者阵营（阵营混编）。地图中央有3个裂隙能量节点，占领节点累积"帷幕充能"，先达到100%的队伍获胜。

## 队伍构成示例

| 位置 | 蚀者方 | 守幕方 |
|------|--------|--------|
| 前排 | 蚀者(近战刺客) | 帷幕守卫(坦克/防线) |
| 控制 | 冥僧人(裂隙引导+堕化) | 草药学者(灵符陷阱+蚀雾) |
| 后排 | 灵痕追猎者(远程输出) | 帷幕学者(侦察+察灵标记) |

## 节点交互

- 占领节点需3s引导，受到伤害打断
- 节点被占领后每2s充能1%
- 蚀者可在节点附近消耗充能进度换取自身临时Buff(+20%伤害，10s)
- 守幕者可在节点附近消耗充能进度换取全队治疗(每人恢复30%灵焰)

# 六、帷幕之域：开放世界（1-4人）

## 六大区域

| 区域 | 旧圣座 | 核心威胁 | 核心秘密 |
|------|--------|---------|----------|
| 永冬山脉 | 霜脊 | 极寒灵焰流失、冰裂隙、冰封蚀者 | 瓦尔德的霜纹石板、霜巨灵群像 |
| 裂霆高地 | 裂霆 | 雷暴随机打击、静电灵痕 | 天穹之冠圆顶密室、三兄弟残响 |
| 幽翠残林 | 翠庭 | 灵质迷雾、根语幻觉、渡鸦群 | 莫丽甘娜的渡鸦冢、圣橡幼苗 |
| 骨灰平原 | 金沙 | 灵焰持续流失、灵符信息过载 | 命符殿入口、天秤碎片、日舟航道 |
| 白石遗址 | 白石 | 律法余音防御、塌方 | 朱沃纳斯的隐藏之字、雅努维翁之门 |
| 雾隐禅院 | 雾霭 | 雾中迷失、错步坠虚空 | 玉穹尊者永恒之茶、沉没仙岛 |

## 生存三轴

- **灵焰管理**：储量(HP)、纯度(蚀痕污染)、温度(活跃度)、完整度(结构)
- **建造系统**：临时篝火→简易营地→加固庇护所(重生点)
- **灵质生态**：灵植采集、灵质矿物开采、古代遗物发掘(四类：灵格/知识/货币/竞技外观)

## 裂隙风暴（随机世界事件）

三阶段：裂隙预警(3-5min)→裂隙撕裂(蚀者增益、环境恶化、10-15min)→裂隙消退(掉落裂隙残渣、灵植高产)

## 角色养成：灵格五大阶段（1-100级）

每阶段解锁新能力槽位和系统权限。满级解锁专属传说灵格——如"独目者的远见"（视野永久+10%）。

## 竞技联动

- 探索获得的古代遗物→解锁竞技模式角色外观
- 灵格等级提供微量竞技属性(排位赛禁用)
- 千面之影真相档案需要全社区共同推进

# 八、建议开发优先级

| 优先级 | 模式 | 理由 |
|--------|------|------|
| P0-立即 | 帷幕追猎(非对称PvP) | 已有Unity基础(PositionSync/PlayerController/VillageScene)，差异化最强 |
| P1-短期 | 裂隙竞技场(对称PvP) | 复用追猎模式的核心系统，开发成本低 |
| P1-短期 | 守幕者之夜(PvE) | 无需匹配等待，适合单人或小团队快速开玩 |
| P2-中期 | 暗幕推理(社交推理) | 需要完整的灵焰视野渲染系统和会议系统 |
| P3-长期 | 帷幕之域(开放世界) | 规模最大，需要六大区域的完整3D资产 |
