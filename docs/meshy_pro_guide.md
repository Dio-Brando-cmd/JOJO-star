# Meshy Pro 角色生成终极指南

> Meshy Pro 权益：500积分/月 | Meshy 6 最高画质 | 多视图 Image-to-3D | 商用授权 | FBX/GLB 导出

---

## 黄金法则：Image-to-3D >> Text-to-3D

Meshy 的 Text-to-3D 靠 AI 从文字猜形状，Image-to-3D 从真实图片提取细节。

**先让 AI 生图，再用 Meshy 把图转成 3D**，质量差 2-3 倍。

```
步骤 1: 用免费 AI 画图工具生角色三视图
        → Leonardo.ai / Stable Diffusion / DALL-E / Midjourney
步骤 2: Meshy → Image to 3D → 上传图片
步骤 3: 下载 FBX → Blender 微调
```

---

## 方案 A：你已有画图工具

如果你能用 Midjourney / Stable Diffusion / Leonardo.ai：

### 先在画图工具里生成角色

用下面的 prompt 生成角色 **正面全身 T-pose**：

```
full body character sheet, front view, T-pose, white background,
[角色详情同下方方案B的prompt],
character design sheet, game asset reference,
photorealistic, even studio lighting, clean edges
```

生成正面一张、侧面一张 → 导入 Meshy **Image to 3D (Multi-View)** → 最高质量。

## Meshy 操作清单

每次生成一个角色：

- [ ] Meshy → **Text to 3D**
- [ ] 引擎选 **Meshy 6**（Pro 才有）
- [ ] Mode 选 **PBR**
- [ ] Style 选 **Realistic**
- [ ] Poly Count 选 **High (50K-100K)**
- [ ] 粘贴 prompt → Generate
- [ ] 等 ~90 秒出 4 个结果
- [ ] 选最好的 → **Download FBX**
- [ ] 如果 4 个都不满意 → 点 **Retry**（不重复扣积分？检查一下，Meshy 可能每次都扣）
- [ ] 每个角色导出后存到 `werewolf_models/` 目录

---

## 后处理（Blender）

Meshy 出来的模型都需要做这几步：

1. **缩放到真实身高** — 导入 Unity 的 1m 立方体做参考
2. **Decimate 减面** → 50K→15K（移动端）或 50K→30K（PC端）
3. **检查材质** → 确保 PBR 三张贴图都在（Base Color / Normal / Roughness）
4. **去 Mixamo 绑骨** → mixamo.com 上传 → 自动骨骼 → 下载带动画的 FBX
