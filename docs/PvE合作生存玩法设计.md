# 帷幕之地 -- 守幕者之夜：5人PvE合作生存玩法设计

> 版本：v1.0 | 目标模式：5人合作PvE | 基于世界观：帷幕之地·灵焰纪元

---

## 一、模式概述与游戏目标

### 模式名称
**守幕者之夜（The Warden's Night）**

### 核心概念
五名玩家扮演守幕者聚落的幸存者，在夜幕降临时共同保护聚落核心的「帷幕圣坛」。每一夜，蚀痕从帷幕裂隙向外渗透，聚落周围的蚀者群被吸引而来，一波接一波地冲击防线。五名守幕者必须协同修复裂隙、击退蚀者、管理有限的灵焰与灵符资源，撑到黎明。

### 胜利条件（主目标）
**在五波蚀者潮中存活并保护圣坛不被摧毁，最终封印本场出现的所有裂隙，迎来黎明。**

### 合作机制
- **分工绑定**：五名玩家分别扮演八个可选职业中的五个，每个职业在PvE中有不可替代的专长。开局大厅设"职业协商"阶段（90秒），玩家亮出自己倾向的职业并协调分工，系统会根据缺少的关键职能给出提示（如"你们缺少愈灵师——这意味着没有稳定的灵焰恢复来源"）。
- **共享灵焰池**：五名玩家的灵焰值汇聚为一个"聚落灵焰池"，最高1500点（初始每人贡献200点，上限可被升级）。当任意玩家受到蚀痕伤害时，优先消耗个人灵焰，个人灵焰耗尽后开始消耗聚落灵焰池。灵焰池归零则全队灵焰尽灭，游戏结束。
- **连携技**：特定职业组合在近距离（15m内）触发连携技。例如帷幕守卫+灵痕追猎者同时瞄准一个目标时，追猎者的灵焰猎枪伤害+50%且守卫自动吸收该次攻击引发的蚀痕反噬。连携技冷却120秒，每个职业对最多拥有2个连携搭档。
- **信息共享**：帷幕学者在地图上标记的裂隙位置会实时同步给全队。灵织者的"帷幕低语"线索以文字气泡形式推送到全队HUD。任何玩家发现的叙事碎片自动解锁到全队的"守幕者笔记"中。
- **倒地救援**：玩家灵焰值归零后不会立即死亡，而是进入"灵焰摇曳"状态（倒地，120秒可被救援）。倒地期间每30秒流失50点聚落灵焰池。任意队友靠近后长按交互键5秒完成救援，被救者恢复50点个人灵焰。若倒地后120秒未被救起，该玩家的灵焰彻底消散，永久出局。

## 二、夜晚波次设计（五波蚀者潮）

游戏时间结构：每波由"准备阶段（45秒）+ 战斗阶段（波次持续直到蚀者清零）+ 间隙阶段（60秒）"组成。五波总时长约25-35分钟。

### 第一波：蚀痕渗透（裂隙初现）
- **蚀者构成**：8-10只普通蚀者（蚀痕鼠、蚀痕爬行者）
- **蚀者特点**：单体弱（每只约100HP），移速中等（3.5m/s），近战攻击每次造成15灵焰伤害。没有特殊能力。
- **裂隙**：地图随机出现1道小型裂隙（不会扩大），每30秒从中涌出2只蚀者，直到被修复。
- **阶段目标**：击杀所有蚀者的同时，让帷幕学者找到并标记裂隙，然后由任意玩家站在裂隙旁引导修复（持续15秒，期间无法攻击，需要队友保护）。
- **设计意图**：教学波次。教会玩家基础战斗、裂隙修复流程、团队配合站位。不会对防线造成真正威胁。

### 第二波：蚀痕变异（蚀者种类多样化）
- **蚀者构成**：6只普通蚀者 + 3只蚀痕射手（远程）+ 1只蚀痕卫士（高防盾怪）
- **蚀者特点**：
  - 蚀痕射手：HP 80，远程抛射蚀痕弹（30m射程，20灵焰伤害），移速慢（2m/s）
  - 蚀痕卫士：HP 300，正面自带70%减伤护盾，移速极慢（1.5m/s），近战攻击40灵焰伤害。弱点是背后（无减伤）
- **裂隙**：1道小型裂隙+1道正在扩大的中型裂隙。中型裂隙每25秒涌出2只蚀者，且裂隙半径15m内所有玩家持续受到蚀痕侵蚀（每秒5点灵焰流失）。
- **阶段目标**：优先击杀蚀痕射手（它们会远程骚扰正在修复裂隙的队友），然后由灵痕追猎者或蚀者（若选了蚀者角色）绕后击杀蚀痕卫士。同时修复两道裂隙。
- **设计意图**：引入多类型敌人和站位策略。玩家必须学会前后排分工。

### 第三波：蚀痕狂潮（数量碾压）
- **蚀者构成**：15只普通蚀者 + 5只蚀痕射手 + 2只蚀痕卫士 + 1只蚀痕尖叫者
- **蚀者特点**：
  - 蚀痕尖叫者（新敌人）：HP 200，到达圣坛10m范围内开始蓄力尖叫（3秒前摇）。尖叫对全场所有玩家造成50灵焰伤害并打断所有正在进行的裂隙修复。尖叫冷却30秒。击杀优先级最高。
- **裂隙**：2道中型裂隙 + 1道大型裂隙。大型裂隙每20秒涌出3只蚀者，且在裂隙20m范围内的地面缓慢生成"蚀痕沼泽"——踏入则移速-40%并每秒流失8灵焰。
- **阶段特殊事件**：第3波战斗开始后60秒，聚落外围一处栅栏被蚀者冲破，圣坛暴露面积增加，蚀者到达圣坛的路径缩短。需要一名玩家前往栅栏缺口，消耗80点聚落灵焰池完成"灵焰修补"（10秒引导）。
- **设计意图**：压力波次。同时考验单体击杀效率、群体控制、目标优先级判断和紧急事件处理。队伍必须分头行动。

### 第四波：冥僧人的进军（精英敌人登场）
- **蚀者构成**：10只普通蚀者 + 3只蚀痕卫士 + 2只蚀痕尖叫者 + 4名冥僧人学徒
- **冥僧人学徒（精英）**：HP 350，拥有独立灵焰意识。能力：
  - 蚀痕链：连接最多2只普通蚀者，被连接的蚀者移速+50%且死亡时产生蚀痕爆炸（范围8m，20灵焰伤害）
  - 灵焰汲取：锁定一名玩家，持续引导（可被打断），每秒吸取10灵焰注入自身（恢复HP）
  - 学徒死亡时留下一个持续30秒的"蚀痕残迹"，站在上面的蚀者每秒恢复15HP
- **裂隙**：3道中型裂隙，且裂隙之间存在"共振"——如果任意两道裂隙之间的距离小于40m且同时处于开启状态，每15秒产生一次"裂隙脉冲"，全图所有蚀者获得5秒的狂暴（攻速+30%，移速+20%）。
- **地图变化**：聚落边缘开始出现"蚀痕藤蔓"，在某些路径上形成减速区域。草药学者的蚀灭符可以烧毁藤蔓（范围清场）。
- **设计意图**：首次引入具有战术意识的敌人。冥僧人学徒会优先链接最靠近圣坛的蚀者，迫使玩家先击杀学徒再处理杂兵，否则爆炸连锁可能瞬间摧毁防线。裂隙共振机制迫使队伍不能慢慢来——必须同时修复相邻裂隙，或先打通一条安全走廊。

### 第五波：帷幕边缘（最终波次 + Boss战前奏）
- **蚀者构成**：12只普通蚀者 + 4只蚀痕射手 + 3只蚀痕卫士 + 2只蚀痕尖叫者 + 3名冥僧人学徒 + 1名冥僧人首领（Boss A）
- **冥僧人首领**：详见第三节Boss战设计。
- **裂隙**：全场裂隙在Boss被击败前不会停止涌出蚀者。每45秒随机出现1道新的小型裂隙（最多同时存在6道）。击败冥僧人首领后，所有裂隙自动停止涌出，但已存在的蚀者不会消失。
- **Boss触发条件**：击杀本波所有3名冥僧人学徒后，冥僧人首领从圣坛正北方的帷幕裂隙中走出。
- **设计意图**：终极防御考验。在前四波积累的资源管理和战术默契将在这里得到最终检验。成功击败冥僧人首领后进入结局判定。

## 三、资源管理系统

### 3.1 灵焰（Spirit Flame）

灵焰是游戏的核心生命资源，同时作为货币使用。

| 项目 | 数值 |
|------|------|
| 个人灵焰上限 | 250点/人 |
| 聚落灵焰池上限 | 1500点（初始1000，可通过升级提升） |
| 倒地流失速率 | 每30秒50点（从聚落池扣除） |
| 救援后恢复 | 50点个人灵焰 |
| 蚀痕伤害 | 侵蚀状态下每秒流失3-15点（视蚀痕等级） |

**灵焰获取方式**：
- 击杀蚀者掉落"灵焰残片"（小怪1-3个，精英5-8个），每个恢复5点聚落灵焰池
- 愈灵师的"灵焰共鸣"技能：站在圣坛旁引导15秒，为聚落灵焰池恢复80点（冷却90秒）
- 帷幕学者成功修复一道裂隙：聚落灵焰池恢复100点
- 波次间隙阶段，聚落灵焰池自动恢复波次数的平方x10点（第一波间隙+10，第二波+40，第三波+90...）

### 3.2 灵符（Spirit Talismans）

灵符是消耗型技能道具。每名玩家拥有独立的灵符携带上限（2枚），波次间隙可在圣坛处制作。

| 灵符名称 | 制作消耗 | 效果 |
|----------|----------|------|
| 蚀灭符 | 30灵焰 | 投掷到目标区域，造成范围80蚀痕伤害（对蚀者特攻），并清除区域内蚀痕沼泽和藤蔓 |
| 愈灵符 | 20灵焰 | 为一名队友恢复60点个人灵焰，或在倒地队友身上使用可远程救援（无需靠近） |
| 帷幕符 | 40灵焰 | 在目标位置生成一个持续20秒的帷幕屏障，阻挡蚀者通行（Boss级蚀者可破坏，需要10秒） |
| 追猎符 | 25灵焰 | 标记一名敌人，全队对其伤害+25%，持续15秒 |
| 净化符 | 35灵焰 | 移除一名玩家身上的所有蚀痕侵蚀效果，并使其在接下来15秒内免疫蚀痕侵蚀 |

每名玩家每波最多制作2枚灵符（间隙阶段在圣坛处进行），且携带上限为2枚。草药学者天赋"灵符专精"可将制作上限提升至3枚、携带上限提升至3枚。

### 3.3 庇护所耐久（Sanctuary Integrity）

圣坛是聚落的心脏。如果圣坛被摧毁，游戏直接以最差结局结束。

| 属性 | 数值 |
|------|------|
| 初始耐久 | 500/500 |
| 蚀者攻击圣坛 | 每次近战攻击消耗15耐久（普通蚀者） |
| 蚀痕侵蚀圣坛 | 若有未修复的裂隙存在，每30秒圣坛自动流失5耐久 |
| 圣坛自我修复 | 波次间隙阶段，圣坛恢复100耐久（前提是至少有一名玩家在圣坛旁使用"灵焰灌注"交互——持续15秒引导，可被中断） |
| 耐久降至0 | 圣坛崩溃，触发最差结局——"帷幕永夜" |

帷幕守卫的"守护誓言"绑定圣坛后，圣坛获得15%减伤。灵织者中的锻甲灵织者可以使用铁锤对圣坛进行一次紧急加固（+50耐久，每局仅限1次）。

## 四、角色分工（8职业在PvE中的定位）

### 4.1 蚀者（CORRUPTED）—— 暗蚀收割者
**PvE定位**：近战刺客 / 蚀痕免疫 / 高爆发

蚀者玩家在PvE中自愿扮演一个"被蚀痕部分侵蚀但未完全堕化"的守幕者。他们体内残存的蚀痕赋予他们特殊能力，但每一次使用都会加剧体内的蚀痕侵蚀。

- **蚀痕斩击**：近战攻击蚀者造成120%伤害，且击杀蚀者后恢复10点个人灵焰。
- **蚀痕护体**：被动免疫蚀痕沼泽的减速和伤害效果。在裂隙20m范围内不会受到灵焰流失。
- **蚀痕爆发（大招，冷却180秒）**：消耗50点个人灵焰，在自身周围10m释放蚀痕冲击波，对范围内所有蚀者造成150伤害并眩晕3秒。使用后自身进入"蚀痕过载"状态——接下来的30秒内受到的所有蚀痕伤害+50%。
- **PvE特有风险**：蚀者玩家每次使用主动技能都会累积"蚀痕深度"。累积至100%时触发"蚀痕失控"——玩家失去控制15秒，变成AI操控的中立敌对方（攻击最近的单位，无论敌友）。失控结束后蚀痕深度归零并造成50点聚落灵焰池穿刺损失。这迫使蚀者玩家在高爆发收益和失控风险之间做出战术权衡。

### 4.2 冥僧人（NETHER_MONK）—— 破面者
**PvE定位**：远程打断 / 裂隙专家 / 控场

冥僧人玩家在PvE中被设定为"叛出冥僧人教团、选择守护人类的异端"。他比其他守幕者更了解帷幕裂隙的运作机制，因而在裂隙修复方面有天然优势。

- **冥想低语**：感知全场所有裂隙的精确位置和扩大速率（为全队提供裂隙预警）。
- **裂隙抑制**：在裂隙15m内引导专注，使该裂隙的涌出速率减半（持续10秒，冷却30秒）。这是唯一能延迟裂隙涌出的能力。
- **灵焰断绝（大招，冷却150秒）**：对一名精英/Boss敌人施加"灵焰标记"，持续20秒。标记期间，该敌人无法被任何方式恢复HP，且所有玩家对其伤害+15%。
- **异端烙印**：对蚀者的击杀伤害+10%，但自身被蚀者攻击时受到的灵焰流失+20%（冥僧人教团视其为叛徒，蚀痕会主动寻找他）。

### 4.3 帷幕学者（VEIL_SCHOLAR）—— 裂隙观测者
**PvE定位**：远程侦查 / 裂隙修复专精 / 全队信息核心

- **裂隙分析**：靠近裂隙后可进行"深度分析"（8秒引导，可被打断），完成后全队对该裂隙附近所有蚀者获得20%伤害加成，持续45秒。
- **学者标记**：标记一名敌人，使其轮廓在全队视野中高亮显示（可穿墙），持续20秒。冷却45秒。
- **帷幕观测（大招，冷却120秒）**：全图感知——揭示地图上所有蚀者的位置（以红点形式出现在小地图上），持续8秒。同时预告下一道裂隙出现的可能位置（以紫色闪烁标记）。
- **知识诅咒**：帷幕学者不能使用任何恢复型灵符（愈灵符对自己无效），且被救援所需时间翻倍（10秒）。但她在倒地状态下仍可使用"帷幕观测"技能——这是一把双刃剑。

### 4.4 草药学者（HERBAL_SAGE）—— 蚀灭专家
**PvE定位**：范围清场 / 灵符制造者 / 区域控制

- **蚀灭符专精**：草药学者制造的蚀灭符伤害+30%（104点），且爆炸范围+25%。
- **蚀痕净化场**：在地面放置一个持续15秒的净化区域（半径8m），站在其中的队友免疫蚀痕侵蚀且每秒恢复5点个人灵焰。冷却60秒。
- **灵符增产（被动）**：草药学者每波可制作3枚灵符（其他职业2枚），携带上限3枚。在圣坛旁制作灵符时，消耗降低30%。
- **最后的蚀灭（大招，冷却200秒）**：投掷一枚大型蚀灭符，对目标区域（半径12m）造成200蚀痕伤害。如果此技能击杀了至少3只蚀者，则冷却时间减半。使用后草药学者在接下来45秒内移动速度-30%（体力透支）。

### 4.5 愈灵师（SPIRIT_MENDER）—— 灵焰守护者
**PvE定位**：治疗 / 复活专家 / 续航核心

- **灵焰愈合**：为一名队友恢复40点个人灵焰（射程20m，冷却15秒）。这是唯一的非道具主动治疗技能。
- **灵焰共鸣**：在圣坛旁引导15秒，为聚落灵焰池恢复80点（冷却90秒）。这是维持长期生存的关键续航手段。
- **灵焰护盾（大招，冷却150秒）**：为全队施加一个吸收50灵焰伤害的护盾，持续20秒。护盾存在期间，受保护者不会被蚀痕侵蚀。
- **牺牲转移（被动）**：当一名队友进入"灵焰摇曳"倒地状态时，愈灵师可选择消耗自身60点个人灵焰来远程完成救援（无需靠近，但需要3秒引导，可被打断）。这是唯一能在激烈战斗中救回远处倒地队友的手段。

### 4.6 灵织者（SPIRIT_WEAVER）—— 聚落之魂
**PvE定位**：全能辅助 / 子类型决定特长 / 团队增益

灵织者是PvE中最灵活的职业，8种子类型各自带来不同增益。开局时灵织者玩家可选择子类型（或随机以获得额外初始灵焰奖励）。

| 子类型 | PvE专长 |
|--------|---------|
| 老兵灵织者 | 概率识破隐身的蚀者（如某些精英蚀者），对小怪攻击有10%格挡率 |
| 行路灵织者 | 移速+15%，救援倒地队友所需时间减半（2.5秒），可同时携带3枚灵符 |
| 学徒灵织者 | 可进行简单治疗（恢复25点，冷却25秒），与愈灵师站在一起时治疗效果+30% |
| 记述灵织者 | 全队小地图视野+30%，在战斗中发现叙事碎片后全队获得短暂伤害加成（+10%，持续60秒） |
| 守夜灵织者 | 小地图上对靠近圣坛20m内的蚀者自动标记红点，陷阱类的蚀痕藤蔓对其可见 |
| 炊火灵织者 | 在圣坛旁时全队灵焰恢复速率+25%，间隙阶段的圣坛自我修复效率+50% |
| 锻甲灵织者 | 圣坛紧急加固（+50耐久/局限1次），对蚀痕卫士和Boss造成的伤害+15% |
| 织幕灵织者 | 帷幕低语线索更精确，裂隙预警时间提前10秒，全队的裂隙修复速度+20% |

- **灵织者的意志（大招，冷却120秒）**：全队伤害+15%、移速+10%、灵焰恢复+5/秒，持续15秒。效果结束后灵织者自身进入30秒疲惫状态（移速-20%，无法使用主动技能）。
- **核心价值**：灵织者是"团队放大镜"——本身不突出，但让其他四个职业的效果都被放大。

### 4.7 帷幕守卫（VEIL_GUARDIAN）—— 不朽之壁
**PvE定位**：坦克 / 防线核心 / 圣坛守护者

- **盾牌格挡**：举盾状态下正面减伤50%，但移速-40%。格挡一次攻击消耗15体力（体力上限100，每秒恢复8点）。
- **守护誓言**：可以绑定一名队友或圣坛。绑定队友时，该队友受到伤害的30%转移到守卫身上；绑定圣坛时，圣坛获得15%减伤。冷却60秒切换绑定目标。
- **盾击**：近战击退前方锥形范围内的蚀者5m并造成50伤害，眩晕2秒。冷却15秒。
- **筑垒（大招，冷却180秒）**：在目标位置筑起一道持续25秒的灵焰壁垒（宽8m），蚀者无法穿越（Boss级蚀者可以破坏，需要10秒）。壁垒同时为站在其后的队友提供20%减伤。
- **银之恐惧（被动）**：帷幕守卫靠近裂隙时，裂隙涌出速率自动-20%（叠加冥僧人的裂隙抑制效果）。

### 4.8 灵痕追猎者（FLAME_TRACKER）—— 蚀者猎人
**PvE定位**：远程单体爆杀 / Boss杀手 / 弱点攻击

- **灵焰猎枪**：远程射击（射程60m），每发造成120伤害。装弹2发，波次间隙在圣坛旁补充弹药。击杀蚀者后，下一发子弹伤害+25%（可叠加，最多+75%）。
- **弱点标记**：瞄准一名敌人3秒后，揭示其弱点部位。攻击弱点造成双倍伤害。冷却60秒。
- **追踪弹（大招，冷却150秒）**：射出一发追踪灵焰弹，自动锁定当前血量最高的Boss/精英敌人，造成250伤害并附加"蚀痕灼烧"效果（每秒15伤害，持续10秒）。此技能无法被闪避或格挡。
- **猎人之怒（被动）**：当场上有队友倒地时，灵痕追猎者的伤害+20%且移速+15%。当场上有队友被救援成功时，灵痕追猎者的下一发子弹必定暴击（双倍伤害，可与弱点标记叠加至四倍）。

## 五、Boss战设计

### Boss A：超级蚀者 —— 「蚀痕巨兽」

**出现时机**：第三波或第四波结束后（取决于队伍在前三波的表现，若修复裂隙数量不足，则提前在第三波结尾出现）。

**外观**：由数十只蚀者融合而成的巨型怪物。高约4米，全身覆盖蠕动的黑色蚀痕物质，不断有小蚀者从它体表分裂出来（诞生时发出刺耳的婴儿啼哭声）。它的核心——一团不断跳动的暗紫色灵焰——半暴露在胸腔中，这是唯一能被有效伤害的部位。

**核心机制**：
- **血量**：3000HP（弱点——胸腔核心承受全额伤害，其他部位减伤60%）
- **能力1 —— 蚀痕喷吐**：向前方锥形区域喷吐蚀痕洪流，造成60灵焰伤害并在命中区域生成持续20秒的蚀痕沼泽。冷却20秒。前摇明显（胸腔核心膨胀并发光），玩家有2.5秒时间侧移躲避。
- **能力2 —— 蚀痕分裂**：每损失25%HP（750点），巨兽分裂出4只普通蚀者和1只蚀痕卫士。分裂后巨兽本体移速-20%持续10秒（体型缩小）。
- **能力3 —— 蚀痕汲取**：巨兽将触须插入地面，从全场所有未修复的裂隙中汲取能量。每道裂隙为它恢复5%最大HP（150点），引导持续6秒，可被打断（攻击胸腔核心即可）。
- **狂暴阶段（HP低于25%）** ：巨兽移速+40%，攻击频率+50%，并持续从体表分裂蚀者（每10秒2只，直到被击杀）。胸腔核心的保护膜破裂，弱点暴露面积增大，所有玩家对其伤害+30%。

**战术要点**：
- 必须有人持续清理巨兽分裂出的小蚀者，否则数量会压倒防线。
- 冥僧人的"灵焰断绝"在此时价值极高——阻断巨兽从裂隙回血。
- 灵痕追猎者的弱点标记是关键输出手段，叠满猎枪连杀buff后一发弱点暴击可造成480伤害。
- 队伍优先修复裂隙，剥夺巨兽的回血来源。

### Boss B：冥僧人首领 —— 「厄尔苏拉斯，第一位破面者」

**背景**：厄尔苏拉斯曾是白石圣座最杰出的灵焰学者，在诸神黄昏后他主动走向帷幕裂隙，试图以自己的灵焰填补裂隙。但他失败了——裂隙的力量侵入他的灵焰，将他扭曲为第一个、也是最强大的冥僧人。如今他统领冥僧人教团，以"通过蚀痕净化世界"为信仰，试图扩大帷幕裂隙让千面之影降临。

**出现时机**：第五波，击杀所有3名冥僧人学徒后从帷幕裂隙中走出。

**外观**：身穿早已褪色的白石圣座学者袍（布满蚀痕灼烧的焦痕），面部被一道纵贯的裂隙状伤疤一分为二——半边脸保持人类形态（苍老、疲惫、眼中仍有学者的执着），另半边脸完全被蠕动的蚀痕遮覆（眼窝中燃烧着暗紫色火焰）。右手化为蚀痕触须，左手仍握着当年用于记录的残缺羊皮卷。他的声音在人类语言和蚀痕的低语之间不断切换。

**核心机制**：
- **血量**：4500HP
- **能力1 —— 学者的审判**：厄尔苏拉斯吟唱一段残缺的诸神契约文本（3秒），然后对全场所有玩家造成35灵焰伤害。如果玩家在吟唱期间处于移动状态，伤害翻倍。这意味着玩家必须在吟唱时停下脚步"倾听审判"来减少伤害——这是一个反直觉的机制：你越害怕、越想跑，受到的伤害越大。冷却35秒。
- **能力2 —— 裂隙召唤**：厄尔苏拉斯在随机两名玩家的脚下各召唤一道临时裂隙（直径5m），裂隙在3秒后爆发，对站在其中的玩家造成50灵焰伤害并将他们击退10m。冷却25秒。
- **能力3 —— 灵焰反转**：厄尔苏拉斯指向一名玩家（有红线提示，持续2秒），该玩家接下来8秒内——所有治疗变为伤害、所有伤害变为治疗。被标记的玩家必须在8秒内停止使用所有主动技能。冷却40秒。
- **能力4 —— 蚀痕风暴（大招，HP低于50%时解锁）** ：厄尔苏拉斯升空并进入无敌状态，全场随机降下蚀痕雨（小范围区域伤害，每秒20灵焰，持续8秒）。同时所有裂隙涌出速率翻倍。蚀痕风暴结束后厄尔苏拉斯落地并进入10秒虚弱状态（承受伤害+30%）。
- **能力5 —— 契约背弃（HP低于15%时触发一次）**：厄尔苏拉斯以最后的人性发出求救："帮我……把它从我体内驱逐出去……"然后进入15秒的"灵焰撕裂"状态——他的蚀痕半身和人类半身互相攻击，造成他自身HP快速流失（每秒100点）。但在此期间，所有靠近他10m的玩家每秒受到30点灵焰伤害。队伍的抉择：靠近他加速击杀（承受伤害）还是远远等他自残完毕（但此时裂隙仍在涌出蚀者）。

**战术要点**：
- "学者的审判"必须由一名玩家提前提示全队停步——这最适合由帷幕学者或灵织者承担指挥。
- "灵焰反转"是最致命的机制——愈灵师如果在中招期间给队友治疗反而会造成伤害。需要愈灵师提前设置一个"我中招了"的快捷信号。
- 在蚀痕风暴期间，全队必须分散寻找安全点，同时确保至少一名玩家在圣坛旁清理趁机靠近的杂兵。
- 契约背弃阶段的近身伤害极高，建议用追踪弹或蚀灭符远程收尾。

### Boss C：千面之影化身 —— 「你不是你」

**背景**：千面之影——幕外最古老的混沌存在——无法以真身进入帷幕。但它可以将自己的一部分投射进来，而投射的形态取决于……观察它的人。每一个守幕者看到的千面之影化身都不一样。它呈现为观察者潜意识深处最恐惧、最愧疚、最无法面对的那个形象。这就是它的名字——「你不是你」，因为它让你看到的不是它，而是你自己最深处的恐惧。

五名玩家会同时看到千面之影呈现为五种不同的视觉形态（由各自客户端独立渲染）。

**出现时机**：仅在地狱结局路径中出现。当第五波结束时圣坛耐久低于30%且聚落灵焰池低于300点，或选择了"直面千面之影"的特殊剧情选择，则触发千面之影化身战。此战为隐藏Boss，不是每局都会遇到。

**外观**（每个玩家看到的不同）：
- 可能看到自己死去的爱人（对帷幕学者而言：可能是她在裂隙中失去的导师）
- 可能看到自己的"黑暗版本"（对蚀者而言：一个完全堕化的自己）
- 可能看到千面之影呈现为冥僧人首领厄尔苏拉斯（如果队伍刚刚击败他——暗示他从未被真正击败）
- 系统提供5种默认视觉模板，匹配8个职业的主题恐惧

**核心机制**：
- **血量**：6000HP（但千面之影不通过HP归零被击败——见下）
- **核心规则 —— 「镜像灵焰」** ：千面之影没有固定形态。在战斗开始时，它复制在场五名玩家中一人的职业能力（随机选择），并以该职业的强化版本使用这些能力。每损失25%HP，它切换到复制另一个玩家的职业。被复制的玩家在镜像持续期间对自己职业的伤害-50%（"你在攻击你自己"）。
- **能力1 —— 镜像复制（被动，每25%HP切换）**：
  - 复制帷幕学者 → 全场裂隙涌出速率+50%
  - 复制愈灵师 → 自身每秒恢复40HP，且为蚀者杂兵恢复HP
  - 复制灵痕追猎者 → 获得远程攻击能力（每次50灵焰伤害，射程50m）
  - 复制帷幕守卫 → 获得50%减伤
  - 复制草药学者 → 在地面生成蚀痕净化场的反色版本——蚀痕污染场（站在其中的玩家每秒流失15灵焰）
  - 复制蚀者 → 获得蚀痕爆发能力
  - 复制冥僧人 → 可以使用"裂隙召唤"
  - 复制灵织者 → 全场蚀者获得+15%伤害和移速光环
- **能力2 —— 低语侵蚀**：千面之影持续发出只有每个玩家能单独听到的低语。每60秒，一名随机玩家被迫做出"心理选择"——两个选项出现在屏幕上（如"你最害怕失去什么？A. 你的队友 B. 你自己"），选择后触发不同效果（两种都不好，但一种比另一种稍好）。这是叙事性的战斗机制，迫使玩家在高压下审视自己的角色。
- **能力3 —— 千面显现（大招，HP低于30%时触发一次）**：千面之影同时复制全场所有五名玩家的职业能力，持续20秒。所有玩家对自己职业的伤害-50%（包括普通攻击和技能）。唯一的对策是——玩家之间互换目标：灵痕追猎者去打被帷幕学者复制的那一侧，帷幕学者去打被追猎者复制的那一侧，以此类推。
- **击败条件 —— 「自我接纳」**：千面之影不能单纯通过HP归零被击败。当HP降至1时，它进入"混沌无形"状态（无敌，持续30秒）。在此期间，五名玩家必须依次到圣坛前进行"灵焰宣誓"——一个15秒的自述仪式，玩家打字或选择预设选项回答一个问题："你是谁？"。如果五名玩家都完成了宣誓，千面之影的HP归零，触发最终击败。如果有任何一名玩家在30秒内没有完成宣誓（倒地、被杂兵打断、或主动放弃），千面之影恢复15%HP并重新进入战斗。每名玩家每局只有一次宣誓机会——失败则该玩家无法再次宣誓，只能靠其余队友完成。

**战术要点**：
- 当千面之影复制愈灵师时，必须有人去打断它的自我治疗（蚀者的蚀痕爆发、草药学者的蚀灭符都可以）。
- 千面显现阶段是DPS check——如果输出不够，建议全员分散保命等20秒过去。
- "自我接纳"阶段是整场PvE的最高潮——30秒内五人依次宣誓，同时仍要防守杂兵和裂隙。需要极高的配合：两人宣誓时，另外三人必须承担起所有战斗压力。
- 千面之影的低语侵蚀中的每个选择都会影响最终结局的细分类别。

## 六、失败条件与多结局

### 6.1 失败条件（立即触发）

| 失败条件 | 触发时机 | 结局 |
|----------|----------|------|
| 聚落灵焰池归零 | 任意时刻 | 「灵焰尽灭」——全体灵焰消散，聚落沦陷 |
| 圣坛耐久归零 | 任意时刻 | 「帷幕永夜」——圣坛崩塌，帷幕裂隙全面扩大，千面之影降临 |
| 五名玩家全部同时倒地 | 任意时刻 | 「全员摇曳」——无人可救援，聚落灵焰池在3分钟内耗尽 |
| 波次间隙无人进行"灵焰灌注" | 连续2个间隙 | 「圣坛遗弃」——圣坛因缺乏维护而缓慢崩溃 |

### 6.2 胜利结局（完成第五波后依据条件判定）

#### 结局一：帷幕黎明（最佳结局）

**触发条件**：
- 击败冥僧人首领（厄尔苏拉斯）
- 击败千面之影化身（若触发）
- 圣坛耐久≥300
- 聚落灵焰池≥500
- 五名玩家全部存活

**叙事**：
第一缕阳光穿透帷幕，照在圣坛的废墟上。守幕者们站在满地蚀痕残渣中，看着远方的帷幕在晨光中缓缓修复——不是愈合，只是暂时的平息。五道灵焰在圣坛周围交织成一圈微弱但坚定的光环。

帷幕的低语变得清晰了一瞬。不是威胁，不是诱惑——而是一种像是叹息的声音，从帷幕的纤维深处传来。也许诸神的意识还残留在帷幕中。也许他们只是太累了，没有力气再说更多的话。

但守幕者们知道——他们赢得了今晚。而每一个被赢得的夜晚，都是一次对千面之影的投票。幕在加厚。缓慢，但确实地在加厚。

_"今晚的帷幕没有破裂。"_ ——全队成就解锁：**守幕者**

---

#### 结局二：余烬之守（中间结局 — 牺牲）

**触发条件**：
- 击败冥僧人首领
- 至少有一名玩家永久出局（灵焰彻底消散）
- 或圣坛耐久≤200但>0

**叙事**：
太阳升起了。但它的光比昨天暗了一些。圣坛上的裂痕无法完全修复，牺牲者的灵焰已经永远融入了帷幕的纤维中。活着的人站在晨光里，感觉到帷幕变得更厚了一些——代价是其中一人的灵焰。

奈莉亚（帷幕学者）在笔记中写道："诸神用生命编织了帷幕。我们也在做同样的事——只不过我们的生命比诸神短，所以我们得更频繁地牺牲。"

被牺牲者的名字被刻在了圣坛基座的石板上——不，不是刻。是被蚀痕蚀刻出来的。就像他们的灵焰被蚀痕吞噬但又被帷幕吸收了一样——某种循环完成了。

_"诸神已死，我们的灵焰就是新的燃料。"_ ——解锁结局成就：**余烬**

---

#### 结局三：蚀痕折中（灰暗结局 — 苟活）

**触发条件**：
- 未触发千面之影化身（圣坛耐久≥30%且聚落灵焰池≥300）
- 但圣坛耐久<300或聚落灵焰池<500

**叙事**：
黎明到来，但守幕者们没有庆祝。他们知道冥僧人首领虽然被击败了，但帷幕上的裂痕比战斗前更多。蚀痕深植于聚落的每一个角落——连圣坛的基石上都能看到细密的黑色纹路在缓慢扩散。

守幕者们可以继续活一段时间。也许一周，也许一个月。然后下一波蚀者潮会来，下一道裂隙会打开，下一次战斗会开始——但那时，圣坛还能撑多久？灵焰还够分给几个人？

这不是胜利。这是延期。但这也许是人类在帷幕之下能期望的最好的结果。不是在光明中胜利，而是在黄昏中坚持。

_"帷幕没有变薄。也没有变厚。我们只是——没有让它变得更薄。这是今晚唯一的胜利。"_ ——解锁结局成就：**拖延长夜**

---

#### 结局四：千面终临（最差结局）

**触发条件**：
- 未击败冥僧人首领（Boss逃脱或在契约背弃阶段未被击杀）
- 或触发千面之影化身后未能击败它

**叙事**：
千面之影的一缕意识成功穿透了帷幕。它没有完全进入——还没到时候——但它的低语从此混入了帷幕的每一道频率中。守幕者们能听到它。在夜晚的寂静中，在裂隙的嗡鸣中，在圣坛的每一次心跳中。

"你们的神把你们关在这里，说这是保护。但保护，不就是另一种囚禁吗？"

你们知道这只是它的谎言。但问题是——你们不能证明它在说谎。

帷幕没有破裂。但它第一次——让五个人同时产生了一个念头：如果帷幕打开，外面会是什么样？

_"它没有进来。但它让我们打开了窗户——只开了一小条缝。那道缝，也许永远也关不上了。"_ ——解锁结局成就：**窗缝**

---

### 6.3 结局后的Metagame影响（跨局继承）

参考3D玩法拓展方案中设定的"深渊结局"元叙事系统：
- **触发千面终临结局**：接下来3局PvE中，所有裂隙涌出速率+10%（永恒的窗缝）。
- **触发帷幕黎明结局**：接下来3局PvE中，圣坛初始耐久+50，聚落灵焰池初始值+100（帷幕的回应）。
- **连续3局触发帷幕黎明**：解锁隐藏叙事碎片——"诸神的低语"，揭示帷幕编织时一个未被记录的细节。

## 七、地图上可交互的叙事碎片

### 7.1 碎片类型

PvE地图（暮色村+周边区域）散布着20个可交互的叙事碎片。每局随机选择8个出现。收集碎片不提供任何战斗增益，但会逐步揭示世界观真相，并在结局后解锁"守幕者笔记"中的对应条目。

### 7.2 碎片列表（精选12个核心碎片）

| 编号 | 位置 | 类型 | 内容摘要 |
|------|------|------|----------|
| 1 | 圣坛基座背面 | 铭文 | "第一夜——瓦尔德林站在这里。他用空眼框看着帷幕，流下一滴泪。那是第一道裂隙被封印的地方。" |

| 7 | 聚落栅栏的缺口处 | 血渍图案 | 一滩早已干涸的暗紫色蚀痕血迹，形状像一只伸展的手。旁边用小刀刻了四个字："不是第一次。" ——暗示这道栅栏曾经破过，而且上次比这次严重得多。 |

| 9 | 草药学者的药圃（地图西北角） | 可拾取物品 | 一株在蚀痕污染土壤中依然顽强生长的银叶草。拾取后显示文字："它在蚀痕中活下来了。不是因为免疫蚀痕——是因为它学会了把蚀痕当作自己的养料。也许蚀者也是这样开始的。不是对抗，是转化。" |

| 12 | 圣坛正上方的天空（仅帷幕黎明结局后可见） | 终局叙事 | 太阳升起后大约10秒，圣坛正上方的天空中出现一道短暂的光纹——六道不同颜色的灵焰交织成一个环形符号，持续约5秒后消散。这是六座圣座的符文：霜脊的淡青、裂霆的银白、翠庭的翠绿、金沙的金黄、白石的纯白、雾霭的淡紫。它们没有被蚀痕污染。它们只是——在安静地发光。然后消失。留给玩家一个无声的问题：如果诸神已死，这道光纹是谁放的？ |

### 7.3 碎片收集与结局联动

- 收集8个碎片以上（任意组合）→ 结局展示额外段落：每人获得一段"你在帷幕中的位置"独白。
- 收集碎片3号+6号+11号 → 触发隐藏叙事"厄尔苏拉斯的真相"——揭示冥僧人首领并非主动叛变，而是被千面之影以"拯救更多人"的谎言欺骗。
- 收集碎片10号（渡鸦羽毛）→ 在千面终临结局中，莫丽甘娜会额外发送一句低语："我没有放弃你们。请你们也不要放弃自己。" ——改变结局的语调但不改变结果。
