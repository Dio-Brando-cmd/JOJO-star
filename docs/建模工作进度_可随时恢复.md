## 四、下次恢复后的操作清单

### 第 1 步：分析已有模型
```
Blender → Scripting → 打开 blender/analyze_model.py
→ 改 FBX_PATH = r"E:\无敌狼人杀文件\芙蕾雅3.fbx"
→ Run Script → 看报告
```

### 第 2 步：继续生成剩余 9 个角色
```
1. 打开 meshy.ai → Text to 3D
2. 引擎选 Meshy 6 → PBR → Poly Count: High
3. 从 docs/meshy_pro_guide.md 复制 prompt
4. 每个角色跑 2 次 → 选最好的 → 下载 FBX
```

### 第 3 步：导入 Blender 精修
```
1. File → Import → FBX
2. 运行 blender/setup_for_sculpting.py（自动加细分+材质预览）
3. Sculpting 工作区 → 修面部/比例
4. 运行 blender/setup_pbr_texturing.py（创建贴图环境）
```

### 第 4 步：画贴图
```
参照 docs/blender_pbr_guide.md
- 皮肤：吸 Meshy 原贴图颜色 → 画 BaseColor
- Roughness 通道：画皮肤/金属/布料粗糙度变化
```

### 第 5 步：绑骨 + 动画
```
1. 导出 FBX
2. 打开 mixamo.com → 上传 → 自动绑骨 → 选动画
3. 下载带动画的 FBX → 导入 Unity
```
