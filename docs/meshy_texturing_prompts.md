## 贴图不满意怎么办

- **太糊** → 改 4K 分辨率重新生成
- **颜色不对** → prompt 里加具体色号如 `skin RGB(200,170,140)`
- **太光滑** → 加 `rough surface texture, matte finish`
- **皮肤像塑料** → 加 `realistic skin pores, subsurface scattering, micro detail`
- **金属没反光** → 加 `polished metal specular highlights, clear reflections`
