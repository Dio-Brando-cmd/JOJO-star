# 帷幕之地 — 3D 角色建模规格书 v1.0

> 引擎目标: Unity 6000 LTS (URP) | 面数预算: 15K-30K tris/角色 | 贴图: 2K PBR

---

## 通用规范

| 参数 | 数值 |
|------|------|
| 模型格式 | FBX (Blender → Unity) |
| 骨骼 | Humanoid Rig (Mixamo兼容) |
| 面数 | 主体15K + 装备5K + LOD 3级 |
| 贴图 | BaseColor + Normal + Metallic/Roughness + AO (2K) |
| 身高范围 | 1.55m - 2.10m (Unity单位) |
| 动画 | Mixamo自动绑定 + 角色特有动画手动K帧 |

## 一、西格德 — 暮色枪骑

| 属性 | 规格 |
|------|------|
| 性别 | 男 |
| 身高 | 1.88m |
| 体型 | 宽肩窄腰，肌肉线条明显但有岁月痕迹 |
| 年龄感 | 50+，面部有伤疤和皱纹 |

**头部**: 灰白短发，络腮胡。左眉有一道纵贯伤疤。蓝灰色眼睛。
**上装**: 深棕色皮甲，肩部有金属护肩(已锈蚀)。内衬亚麻灰色衬衣。
**下装**: 深色皮裤，膝盖有金属护膝(旧)。
**脚部**: 高筒皮靴，靴底磨损。
**武器**: 猎枪(木托+铁管，枪托上有刻痕计数)——背在身后。腰间短火铳。
**特殊**: 左手无名指有一枚旧婚戒。颈部挂着一枚狼牙吊坠。

**建模要点**: 盔甲需要做旧处理(Normal贴图中加划痕)。猎枪是核心道具，需要较高面数。

## 二、芙蕾雅 — 草药学者

| 属性 | 规格 |
|------|------|
| 性别 | 女 |
| 身高 | 1.65m |
| 体型 | 纤细修长，手部精细(草药师的手指) |
| 年龄感 | 28-32，面部柔和但眼神锐利 |

**头部**: 银灰色长发(及腰)，两侧编有细辫。灰绿色眼睛，眼尾微挑。
**上装**: 深绿色长袍，领口有银色刺绣(草药图案)。腰间系一条皮制工具带。
**下装**: 深棕色长裙，侧面开衩便于行走。
**脚部**: 软皮短靴。
**配饰**: 腰间挂4个小药瓶(绿/蓝/红/紫)。左腕戴一串干花手链。
**特殊**: 颈部有一个不起眼的吊坠——里面藏着她失踪爱人的画像。

**建模要点**: 药瓶需要半透明材质+液体着色。长发需要简单的骨骼链用于物理摆动。

## 三、莫莉安 — 最后的守林人

| 属性 | 规格 |
|------|------|
| 性别 | 女 |
| 身高 | 1.72m |
| 体型 | 瘦削但结实，手臂有肌肉线条(攀爬/采集) |
| 年龄感 | 35-40，脸上有雀斑和日晒痕迹 |

**头部**: 红褐色卷发(及肩)，常戴一顶槲寄生编织的头环。深绿色眼睛。
**上装**: 多层亚麻布料——外层是树皮色斗篷(带兜帽)，内层是苔绿色短衣。
**下装**: 棕色绑腿长裤，膝盖处缝有皮补丁。
**脚部**: 手工皮凉鞋+绑腿。
**配饰**: 脖子上挂着一颗发光的小种子(圣树种子吊坠)。腰间挂收集袋。
**特殊**: 右手掌心有一道绿色的脉络(圣树之力)——低面数但需要发光shader。

**建模要点**: 斗篷的布料模拟很重要。种子吊坠需要自发光材质(Emissive)。

## 四、卡赫特 — 冥界之眼

| 属性 | 规格 |
|------|------|
| 性别 | 男 |
| 身高 | 1.82m |
| 体型 | 瘦削，关节突出，像一个长期斋戒的僧侣 |
| 年龄感 | 难以判断——皮肤光滑但眼神古老 |

**头部**: 光头无发。眼窝深陷。眼珠是罕见的琥珀金色。面部有对称的黑色几何纹身(额心一只眼睛图案)。
**上装**: 黑色长袍，领口高耸。袍子边缘有金色几何纹饰。腰间一条金色腰带。
**下装**: 同色长袍，拖地。
**脚部**: 黑色软鞋。
**配饰**: 右手持一根细长的黑色权杖(顶端是天平形状)。左手腕戴一圈黑色念珠。
**特殊**: 在他注视的方向，空中会短暂浮现一只半透明的金色眼睛(阿努比斯之眼)——用作能力特效而非建模。

**建模要点**: 袍子的褶皱和金色纹饰是视觉重点。纹身用贴图表现。天平权杖需要独立面数预算。

## 五、赫克托 — 不灭之壁

|------|------|
| 性别 | 男 |

| 体型 | 魁梧健壮，肩宽胸厚，摔跤手体型 |
| 年龄感 | 32-35，面容端正，有英雄气 |

**头部**: 黑色短发(卷曲)，方下巴。深棕色眼睛，目光坚定。
**上装**: 青铜色胸甲(有凹痕——挡过致命一击)。内衬白色短袖。

**建模要点**: 金属材质需要Reflection Probe支持。盾牌是核心视觉元素，需要高面数。

## 六、罗慕路斯 — 双重之影

| 属性 | 规格 |
|------|------|
| 性别 | 男 |
| 身高 | 1.78m (人形态) / 2.05m (狼形态) |
| 体型 | 精瘦结实，动作敏捷 |
| 年龄感 | 25-30，面容年轻但有野性 |

**头部**: 深棕色凌乱长发(及肩)。金色眼睛。右耳上方有一小撮灰白色头发。
**人形态上装**: 深灰色皮背心，内衬亚麻衬衣。右臂有狼头纹身。
**人形态下装**: 深色马裤+皮靴。
**特殊**: 在月光下，他的影子偶尔会短暂呈现狼的轮廓——一个shader特效即可。
**狼形态**: 深灰色皮毛，金色眼睛。体型比普通狼大一圈，但比哈尔瓦德的狼形态更精瘦。保留人类时的金眼特征。

**建模要点**: 需要两套模型(人形+狼形)。变身过渡可以用粒子特效遮盖。纹身需要贴图。

## 七、哈尔瓦德 — 银链囚徒

| 属性 | 规格 |
|------|------|
| 性别 | 男 |
| 身高 | 1.92m |
| 体型 | 肌肉膨胀但不匀称，像被力量扭曲了身体 |
| 年龄感 | 28-33，面部有长期痛苦的痕迹 |

**头部**: 黑色乱发(如鬃毛般)。眼睛是暗红色(狼瞳)——低光环境下会微微发光。
**上装**: 破损的深色皮甲(多处爪痕——来自自己的爪子)。不穿内衬。
**下装**: 深色破布长裤，腰间一条粗铁链。
**脚部**: 赤脚或裹布。
**核心道具**: 脖子上系着一条发光的银链(银之恐惧)——银链需要金属shader+微弱的自发光。银链是他自己锁上的。
**配饰**: 双手腕有被锁链长期磨损的伤痕(用Normal贴图表现)。
**特殊**: 在他的狼形态下，银链依然挂在脖子上——提醒他野兽的身份。

**建模要点**: 银链是最重要的视觉元素——需要单独的骨骼用于物理摆动。爪痕在PBR贴图中表现。肌肉线条需要强调。

## 八、斯卡蒂 — 静默猎手

| 属性 | 规格 |
|------|------|
| 性别 | 女 |
| 身高 | 1.75m |
| 体型 | 运动员型——肩宽，手臂有肌肉线条，腿长 |
| 年龄感 | 26-30，面部冷峻少表情 |

**头部**: 白色短发(非常短，几乎是板寸)。冰蓝色眼睛。左脸颊有三道浅色划痕。
**上装**: 白色皮草披肩(从雪山带来的)。内衬深蓝色紧身衣。
**下装**: 白色皮裤，膝盖有护膝。
**脚部**: 白色皮靴(软底，移动无声)。
**武器**: 复合弓背在背后，箭袋在腰间。腰间一把猎刀。
**特殊**: 她的呼吸在空气中凝结成雾气——可以用粒子特效表现。在雪地/雾天环境中她的轮廓会变得模糊。

**建模要点**: 皮草(披肩)用毛发shader或者alpha clip贴图。白色配色需要注意不要过曝。弓是核心道具。

## 九、虚舟 — 缄默之笔

| 属性 | 规格 |
|------|------|
| 性别 | 男 |
| 身高 | 1.70m |
| 体型 | 单薄瘦弱，像一个长期行走的旅人 |
| 年龄感 | 45-55，面容平静如古井 |

**头部**: 剃发(光头)，面部消瘦。深棕色眼睛——很少眨眼。额头有三道横向皱纹。
**上装**: 灰色粗布僧袍(多层)，领口露出墨色内衬。
**下装**: 同色长袍及地。
**脚部**: 草鞋。
**配饰**: 腰间挂一卷宣纸+一支毛笔。左肩背一个布袋(装墨水)。
**核心道具**: 他的宣纸。当他书写时，纸上会出现发光的文字——用Unity UI或贴花系统表现。有时他会把写好的俳句撕下来贴在墙上。
**特殊**: 他不说话——但他的宣纸会"说话"。这是他唯一的沟通方式。

**建模要点**: 布料褶皱是视觉重点。笔和纸是核心道具。僧袍需要简单的布料物理。

## 十、布丽吉德 — 持火者

| 属性 | 规格 |
|------|------|
| 性别 | 女 |
| 身高 | 1.63m |
| 体型 | 小巧但精悍，手臂肌肉来自长期持火把 |
| 年龄感 | 22-26，面部年轻但眼神老成 |

**头部**: 火红色长发(及腰，经常扎成单马尾)。绿色眼睛。面部有淡淡的雀斑。
**上装**: 白色短衣+深红色短斗篷。右手臂缠着防火布料。
**下装**: 深棕色短裤+防火布绑腿。
**脚部**: 棕色短靴。
**核心道具**: 右手手持一根金属火把——火焰永不熄灭。火把是她的核心视觉特征，需要粒子特效+动态光照。
**特殊**: 右手掌心有一道永远不会愈合的烧伤疤痕——她用这只手捧起过圣火的最后一丝火星。疤痕微微发光(自发光Emissive)。
**配饰**: 腰间挂一个火石袋。

**建模要点**: 火焰是核心特效——URP的Particle System + Light。头发颜色需要鲜艳但不刺眼。火把需要金属材质+发光。
