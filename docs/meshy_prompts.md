## 使用技巧

1. **Meshy** 里选 **Text to 3D**，粘贴 prompt
2. 如果效果不好，加关键词：`detailed face, realistic proportions, clean topology`
3. 生成后点 **Download** → 选 **FBX** 格式
4. 不满意就点 **Retry**（每次生成4个候选，选最好的）
5. 下载后导入 Blender：File → Import → FBX
6. 如果要 PS 风格，把 `stylized fantasy art style` 换成 `realistic PBR game asset`
