# 帷幕之地 — 3D 角色建模实操指南

> 从 Blender 到 Unity 的完整工作流

---

## 一、首次设置（一次性）

### 1. 打开 Blender → 运行自动生成脚本

```
1. 启动 Blender 4.2
2. 顶部菜单: Scripting (切换到脚本工作区)
3. 左侧 Text → Open → 选择:
   unity-client/blender/generate_characters.py
4. 点击上方 ▶ Run Script 按钮
5. 等待 15 个角色基础模型生成完毕
6. 输出路径: C:\Users\Lenovo\Desktop\Werewolf_Models\
```

这会自动生成 15 个 FBX 文件，每个包含：
- Mixamo 兼容的人形骨架（20 根主要骨骼）
- 基础身体网格（高度/体型按规格书参数）
- 四肢和头部

### 2. 上传基础模型到 Mixamo → 自动绑定 + 动画

```
1. 浏览器打开 mixamo.com → 登录 Adobe 账号
2. 点击 Upload Character → 选择 FBX 文件
3. 等待自动绑定完成（通常 10-30 秒）
4. 在动画标签页选择需要的动画 → Download
   - 设定: FBX, With Skin, 30fps
5. 重复对所有 15 个角色
```

**必须下载的基础动画（免费）：**
- Idle (待机)
- Walking (走路)
- Running (跑步)
- Crouching (蹲伏)
- Death (死亡)
- Hit Reaction (受击)
- Punching (攻击)

### 3. 在 Blender 中细化模型

Mixamo 绑定后的 FBX 导回 Blender：
- 添加角色特有服装（Block-out 阶段用简单几何体代替）
- 添加标志性道具（猎枪/盾牌/药瓶/火把等）
- 调整面部特征
- 烘焙材质

### 4. 导入 Unity

```
1. 把细化后的所有 FBX 放到一个文件夹
2. Unity 菜单: Tools → Werewolf → Import All Characters
3. 选择 FBX 文件夹 → 自动导入 + 设置 Humanoid Rig + 创建 Prefab
4. Tools → Werewolf → Setup Project (Full) → 一键配置整个项目
5. Tools → Werewolf → Setup Village Scene → 生成村庄场景
```

## 二、快速原型方案（今天就能跑）

如果你现在就想看到角色在场景里动，跳过精细建模：

### 方案 A: Unity 占位胶囊体（5分钟）

```csharp
// 不需要任何模型！Unity会自动用不同颜色的Capsule表示每个角色
// NetworkManager连接服务器后,GameManager3D会自动生成不同颜色的胶囊体
```

打开 Unity → 运行 Setup Project → 点击 Play → 连接服务器 → 看到彩色胶囊体代表玩家。

### 方案 B: Mixamo 快速角色（30分钟）

1. Mixamo.com → Characters 标签页
2. 选择一个免费角色（如 X Bot, Y Bot, Michelle 等）
3. 下载带 Walking/Running/Idle 动画的 FBX
4. 导入 Unity → 作为临时角色使用
5. 15个不同外观的角色可以用 Mixamo 上的免费角色替代
