## 二、3D核心玩法：非对称追逃之夜

### 2.1 玩法概述

**一句话总结**：夜晚不再是回合制按钮点击，而是一场在3D暮色村中展开的、狼人与人类之间的沉浸式追逃游戏。

- **白天**：保留经典桌游模式的讨论/投票（社交推理核心不变）
- **夜晚**：切换到3D追逃模式——狼人自由猎杀，人类躲藏/逃脱/反击
- **核心体验**：类似《黎明杀机》的紧张感 + 狼人杀的身份博弈

### 2.2 夜晚3D流程（120秒）

```
┌─────────────────────────────────────────────────┐
│  夜晚开始 → 全员在自己屋内（门初始锁定3秒）    │
│                                                 │
│  【狼人阵营】                                    │
│  ├─ 自由移动（夜视40m，可冲刺）                 │
│  ├─ 破门进入他人屋子（需要3秒破门动画）         │
│  ├─ 近战攻击（2m范围，冷却8秒）                 │
│  ├─ 嚎叫召集（定位其他狼人，冷却15秒）          │
│  └─ 嗅觉追踪（看到人类留下的踪迹，被动）        │
│                                                 │
│  【人类阵营】                                    │
│  ├─ 自由移动（夜视15m，可冲刺但留踪迹）         │
│  ├─ 锁门/筑垒（守卫可加固任意门）               │
│  ├─ 藏匿（柜子/床底/树丛/水井，Q键）            │
│  ├─ 设陷阱（猎人/毒巫可设陷阱）                 │
│  ├─ 使用能力（按角色不同——详见第五节）         │
│  └─ QTE逃脱（被攻击时500ms窗口）               │
│                                                 │
│  【信息层】                                      │
│  ├─ 离开屋子 → 服务端广播"XX出门了"给邻居       │
│  ├─ 被攻击 → 全地图听到尖叫（方向模糊）         │
│  ├─ 死亡 → 全图广播死亡位置+遗言(可选)          │
│  └─ 村民偷听 → 靠近屋子听到AI生成的环境音        │
└─────────────────────────────────────────────────┘
```

### 2.3 关键设计决策

#### A. 门与房屋系统
```
每个屋子有：
- 前门（可锁/可破/可加固）
- 后窗（可翻越但发出声响）
- 内部藏匿点（床底/衣柜/地下室——每屋2-3个）
- 房主专属物品（提供一次性优势，见第五节）

狼人破门：
- 基础破门时间：3秒（有动画+音效）
- 铁匠加固的门：6秒
- 守卫筑垒的门：无法进入（银之恐惧触发）
- 破门期间狼人无法移动，人类可从后窗逃跑

人类锁门：
- 默认门自动锁（需要狼人破门）
- 出门后门保持解锁状态（风险！）
```

#### B. 踪迹系统
```
人类移动时留下踪迹：
- 走路：轻微踪迹（30秒消散）
- 冲刺：明显踪迹（60秒消散）
- 藏匿：不产生新踪迹
- 雨夜：踪迹加快消散

狼人"嗅觉追踪"能力可看到踪迹（类似发光脚印）
西格德（莽撞特质）：必定留下无法消散的踪迹
斯卡蒂（猎手步伐）：走路时不产生踪迹
```

#### C. 声音传播系统
```
声音半径：
- 破门声：全图可闻（但不暴露位置）
- 尖叫声（被攻击）：40m半径
- 奔跑脚步声：20m半径
- 走路脚步声：10m半径
- 开门声：15m半径
- 嚎叫：全图可闻（暴露狼人位置给所有狼人）
- 枪声：全图可闻（暴露猎人位置给所有人）
```

#### D. QTE逃脱系统
```
被狼人攻击时触发QTE：
- 时间窗口：500ms（基础）
- 逃脱概率：30%（基础）
- 影响因素：
  ├─ 芙蕾雅（纤弱）：窗口缩短到300ms
  ├─ 斯卡蒂（猎手步伐）：+10%逃脱率
  ├─ 西格德（老兵之躯）：被击杀→重伤（15%）
  ├─ 赫克托（英雄之躯）：守护姿态下+25%
  └─ 距离最近藏匿点越近：+逃脱率（梯度加成）
```

## 三、扩展玩法模式

### 3.1 已有：经典桌游模式（BOARD_GAME）
保留现有的回合制夜晚+白天讨论流程，适合快速游戏和移动端。

### 3.2 已有框架：3D追逃模式（THIRD_PERSON）
代码中已定义但未完全实现。完整实现在下文。

### 3.3 🆕 生存之夜（SURVIVAL_NIGHT）

```
核心机制：
- 所有狼人初始觉醒（知道自己身份+同伴）
- 所有人类不知道自己身份（盲人模式）
- 人类需要通过3D环境中的线索推断身份：
  ├─ 在自己屋子里找到"身份线索纸条"
  ├─ 观察他人行为（谁在追谁？谁在帮谁？）
  ├─ 在地标处找到预言家留下的"查验石碑"
  └─ 捡到他人掉落的物品推断身份
- 没有白天讨论——一切靠3D环境中的互动
- 适合5-8人的快速局（10-15分钟）

身份线索示例：
- 猎人的屋子：墙上挂着猎枪架（但枪不见了=猎人带走了）
- 毒巫的屋子：桌上放着半空的药瓶
- 狼人的屋子：床下有沾血的布条
```

### 3.4 🆕 仪式召唤（RITUAL）

```
核心机制：
- 新增第三方阵营：邪教徒
- 村民广场有5个仪式点（对应5个地标）
- 邪教徒可以在仪式点进行献祭（需要30秒引导，可被打断）
- 完成3个仪式 → 邪教徒胜利（召唤帷幕外的存在）
- 人类和狼人都需要阻止邪教徒（但他们互相不知道谁是谁）

地图变化：
- 随着仪式完成，地图发生变化：
  ├─ 1个仪式：月光变红，所有人夜视范围-30%
  ├─ 2个仪式：迷雾加重，地图上随机出现"裂隙"（踏入则受伤）
  └─ 3个仪式：游戏结束，邪教徒胜利动画
```

### 3.5 🆕 猎巫审判（WITCH_HUNT）

```
核心机制：
- 1个玩家是"被诅咒者"（随机，可能是任何人，包括狼人）
- 被诅咒者的存在会导致随机事件：
  ├─ 每60秒，一名随机玩家受到10点伤害
  ├─ 地图上出现"诅咒领域"（踏入减速）
  └─ 破门时间减半（帷幕在变薄）
- 所有玩家需要投票决定谁是"被诅咒者"
- 如果投票正确 → 被诅咒者死亡，游戏恢复正常
- 如果投票错误 → 无辜者死亡，诅咒加强

特殊规则：
- 狼人不能刀被诅咒者（诅咒保护）
- 被诅咒者知道自己被诅咒（但不一定是狼人/邪教徒）
- 这层身份独立于狼人/人类身份
```

### 3.6 🆕 诸神的试炼（TRIAL_OF_GODS）

```
核心机制：
- PvE合作模式（5名玩家 vs AI狼人群）
- 玩家都是人类（无内鬼），需要：
  ├─ 在120秒夜晚中存活
  ├─ 收集散布在地图上的"神遗碎片"
  ├─ 将碎片带到广场古树进行"充能"
  └─ 充能到100% → 人类胜利
- AI狼人由BotManager控制（代码已有）
- 难度递进：每波更多/更强/更聪明的AI狼人

难度等级：
- 第1波：2只普通AI狼人
- 第2波：3只AI狼人 + 1只种狼（可感染）
- 第3波：4只AI狼人 + 雾夜（视野-50%）
- Boss波：芬里尔化身（大型狼，需合力击败）
```

### 4.3 房屋内部设计

每栋屋子有3个房间：前厅 + 卧室 + 地下室/阁楼

```
前厅：
├─ 正门（可锁/破/加固）
├─ 窗户（可翻越但发出声响）
├─ 壁炉（布丽吉德可以点燃——阻止该屋主出门）
└─ 桌椅（可推动制造障碍物）

卧室：
├─ 床（藏匿点1：床底）
├─ 衣柜（藏匿点2：柜内）
├─ 房主专属物品（一次性消耗品）
└─ 后窗（第二个逃生口）

地下室/阁楼：
├─ 秘密藏匿点3（最难被发现）
├─ 身份线索（仅在生存之夜模式）
└─ 可能通往其他屋子的秘密通道（随机生成）
```

### 4.4 动态环境系统

```
天气系统（随机，每局不同）：
├─ 晴天：默认状态
├─ 雾天：所有人视野-40%，踪迹消散速度×2
├─ 雨夜：脚步声-50%可闻范围，但移动速度-10%
├─ 暴风雪：室外持续扣体力，屋内安全
└─ 血月：狼人攻击力+25%，但所有人类获得"第六感"提示

昼夜过渡：
├─ 黄昏（前30秒过渡期）
├─ 深夜（中间60秒，效果最强）
└─ 黎明（最后30秒，人类视野逐渐恢复）

随机事件（每局随机触发1-2个）：
├─ 帷幕裂隙：地图随机出现一个裂隙，靠近会受到10点伤害
├─ 古树低语：广场古树发出低语，给予随机一名玩家线索
├─ 狼群突袭：地图边缘出现AI控制的狼群（PvE干扰）
├─ 神遗显现：地标发光，揭示附近所有玩家的模糊位置
└─ 墓地苏醒：南山墓地随机"复活"一名已死亡玩家为幽灵（可观察不可交互）
```

## 六、背景故事扩展

### 6.1 🆕 新角色：帷幕的意志 — "欧姆尼娅"

```
起源：帷幕本身并非死物。诸神以生命为代价编织帷幕时，
一个微弱的意识在帷幕中诞生了——一个由众神残留意念汇聚而成的存在。
她没有名字，但暮色村的人称她为"欧姆尼娅"（Omnia，意为"一切"）。

每局游戏开始时，所有玩家听到的"指引低语"就是她的声音。
她不是善意的，也不是恶意的——她只是好奇。
她想看看人类在面对恐惧时会做出什么选择。
每次重复的狼人杀循环，都是她的实验。
每次不同的结局，都是她的数据。

她的终极目的无人知晓——
也许是在收集足够的数据来修补帷幕，
也许是在挑选值得活下去的人类，
也许只是孤独——一个有意识的牢笼，在寻找同伴。
```

### 6.2 🆕 幕后Boss：帷幕之外的 "古老存在"

```
在帷幕之外，游荡着那些比诸神更古老的存在。
它们没有名字——或者说，它们的名字本身就是力量，
被诸神从历史中抹去以防召唤。

但当诸神黄昏后，诸神的封印减弱了。
其中一个存在——被称为"千面之影"——
正在试图通过影响帷幕之地的游戏来进入这个世界。

它的策略是：
- 通过"指引"低语中的微妙偏差，影响玩家的决策
- 在血月之夜获得更强的干预能力
- 收集玩家的恐惧/背叛/牺牲作为"钥匙"
- 如果太多玩家在游戏中做出"不道德"的选择（出卖同伴/背后捅刀），
  帷幕就会变薄，千面之影就更接近实体化

这个设定让每一局游戏都有了"元叙事"层面的意义：
玩家之间的背叛不仅仅是为了胜利——它在喂养一个想要毁灭一切的古老存在。
```

### 6.3 🆕 地图叙事碎片系统

```
在3D地图中散布"叙事碎片"——可拾取的环境叙事元素：

类型：
├─ 旧日记残页（文字叙事，拾取后在UI中阅读）
├─ 血渍图案（视觉叙事，暗示过去的狼人杀循环结果）
├─ 回音低语（音频叙事，在特定地标触发）
├─ 物品残留（猎人的空弹壳/女巫的空药瓶/狼人的断爪）
└─ 死者名单（刻在古树上的过去所有参与者的名字——包括当前玩家之前的"前世"）

收集全部碎片的玩家解锁"真相结局"——但真相可能不是他们想看到的。
```

### 6.4 🆕 角色间隐藏关系网

```
现有代码中已有"relationships"字段，3D模式可触发特殊互动：

发现型关系：
├─ 西格德 <-> 哈尔瓦德：世仇（西格德攻击哈尔瓦德时伤害+20%）
├─ 芙蕾雅 <-> 罗慕路斯：罗慕路斯可能是芙蕾雅失踪爱人的线索
├─ 莫莉安 <-> 虚舟：同为各自传统的最后传人（靠近时互相感知）
├─ 布丽吉德 <-> 赫克托：圣火守护者与盾之守护者（天然同盟）
└─ 卡赫特 <-> 哈尔瓦德：卡赫特能看到哈尔瓦德体内的狼形阴影

触发方式：
- 两名存在关系的角色在3D空间中靠近时，双方屏幕出现特殊提示
- 完成"关系任务"（如西格德击杀哈尔瓦德）可获得永久灵格加成
- 未完成时获得叙事性反馈（如"你感觉到和某某之间有什么未完的事情…"）
```

### 6.5 🆕 深渊模式 — 第五种游戏结局

```
当前只有两种结局：好人胜/狼人胜。
新增"深渊结局"：

触发条件：如果游戏进行了超过5轮还没有结束，
且死亡玩家数量≥存活玩家数量×2（总死亡数过多），
则触发"帷幕破裂"事件：

表现：
- 天空裂开，月亮变成黑色
- 所有存活玩家被裂隙拉到同一个空间——"最后审判"
- 存活玩家必须投票决定：牺牲一人来重新封印裂隙，还是大家同归于尽
- 如果牺牲 → 被牺牲者死亡，帷幕修复，游戏继续（好人/狼人正常判定）
- 如果拒绝 → 全体死亡，深渊结局——"千面之影"的封印削弱（永久影响下5局游戏）

深渊结局的meta影响：
- 每触发一次，接下来5局中"随机事件"的概率+5%
- 累积触发3次 → 解锁隐藏角色"欧姆尼娅的化身"（可在特定条件下选择）
```

## 八、立即可行的第一步

基于现有代码，3D追逃模式的最佳起点是：

```

        → 所有存活玩家同时进入3D自由移动
```
