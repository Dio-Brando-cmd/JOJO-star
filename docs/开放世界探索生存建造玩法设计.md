# 帷幕之地 -- 开放世界探索与生存建造玩法设计

> 设计目标：在现有非对称竞技（狼人杀+3D追逃）核心玩法之外，构建一个可单人/合作的开放世界探索与生存建造模式。玩家以暮色聚落为基地，向外探索六座已死圣座的残骸拼接而成的帷幕之域，在灵焰管理、蚀痕抵抗、资源采集、营地建造的循环中逐步揭开世界真相，并将探索所得带回竞技模式使用。

---

## 一、世界探索：六大区域设计

### 1.1 永冬山脉（霜脊圣座遗址）

**地貌特征：** 锯齿状冰峰、永冻冰川、深不见底的冰裂隙、被冰封的远古巨兽骸骨。山脉主体由玄冰和黑玄武岩构成。海拔越高，灵质残留密度越大，但环境也越致命。冰层中偶尔能看到被冻结的霜纹碎片发出的微光。

**环境威胁：**
- 极寒侵蚀：在室外停留超过一定时间，灵焰会开始缓慢流失热量。玩家需要寻找天然温泉（由地下神力残响加热）、点燃随身携带的灵焰暖石、或搭建临时篝火来维持体温。灵焰温度降至临界值以下时，移动速度和交互速度显著下降。
- 冰裂隙：地表随机分布的半透明冰裂隙，踏上去会塌陷，掉入深层冰洞。冰洞底部可能有稀有资源，但需要寻找出口或消耗攀登工具才能脱出。
- 冰封蚀者：永冬山脉的极寒环境中封存着被冻住的大量蚀者——它们在诸神黄昏后进入这片区域觅食时被寒气困住了。一旦玩家在附近释放大量灵焰热量（如使用愈灵能力或大型篝火），冰层可能融化，蚀者会苏醒。

**可发现的秘密：**
- 瓦尔德林最后的霜纹石板：深藏于一个会随月相和极光角度改变位置的冰洞中。玩家需要找到并解读分布在永冬山脉各处的六块"霜纹定位碎片"——每块碎片刻有一道霜纹的不同段落——拼合后可计算出冰洞的当前位置。石板记载着瓦尔德林献祭前最后一条未说出的智慧，解锁后可获得一项永久灵格被动能力（如"独目者的远见"——视野范围永久+10%）。
- 霜巨灵冰封群像：山脉深处一处巨型冰崖上冻结着最后一支霜巨灵族群的全体成员。在特定月光角度下，冰层变得透明，可以看到他们胸中缓慢搏动的灵焰。与群像交互可触发一个支线任务——"等待之人"。
- 虹桥尽头的守望哨：赫尔米昂曾经站立的虹桥尽头遗迹。桥身已大部分坠毁，但尽头处有一块"金瞳水晶"——赫尔米昂在三万天守望中，他的金瞳在冰晶上留下了一道永久性的灵质刻印。拾取后可在竞技模式中解锁"守望者之眼"外观（角色眼睛呈现金色灵光效果）。

**灵植与资源：**
- 霜星苔（普通）：生长在永冻土层裂隙中的银白色苔藓，是愈灵汤剂的基础原料之一。只需要蹲伏采掘，无特殊危险。
- 焚霜苔（稀有）：霜星苔的深红色变种，只生长在零下环境且有神力残响的区域。采集时需要先用灵焰感知定位（它在灵质层面会发出微弱的脉冲信号），然后用特殊容器保存（常温下会自燃）。
- 永冻铁矿石：分布在冰层深处的矿脉，矿石表面永远覆盖着一层不会融化的霜。是锻造灵质门锁和护甲的关键材料。
- 冰晶灵髓（极稀有）：只在永冬山脉最深处的冰洞中发现的液态物质——据说是瓦尔德林那滴落在虹桥上的泪与冰晶混合后形成的。饮用后可永久提升灵焰最大储量一小段。每个玩家角色终生只能使用一次。

**区域NPC：**
- 洛根（霜脊遗民猎手）：住在山脉南麓一间用巨兽骸骨搭建的小屋中，自称是西格德家族的一个远支后裔。他追踪了一头"灵质冰原狼"数年，但从未成功捕获。他可以为玩家提供永冬山脉的详细地图（标注安全路线和裂隙位置），作为交换，玩家需要帮他找到并带回冰原狼的足迹样本。完成他的任务链最终奖励是一把改良猎枪（可在竞技模式中使用——猎枪伤害+5%）。
- 冰语者艾拉：一位在冰洞中独居了二十年的老灵视者。她能从冰晶的纹理中"读"出封存在其中的远古记忆。她正在研究霜巨灵族群的历史，需要玩家深入冰裂隙为她取回特定年代层的冰晶样本。她的任务线最终解锁"霜巨灵的苏醒"——山脉中某个区域会随机出现一头友好的霜巨灵NPC，可以用稀有资源与它交换冰晶灵髓。

### 1.2 裂霆高地（天霆圣座遗址）

**地貌特征：** 巨大裂谷纵横交错、嶙峋岩峰拔地而起、峡谷间由风化形成的天然石桥连接。最高处是泽法利昂的天穹之冠遗址——那座不再被闪电环绕但仍完好无损的银色圆顶巨殿。地面随处可见泽法利昂抛出雷矛后留下的焦痕——每一道焦痕的形状都独一无二，蕴含着一段被雷霆记录的过去。

**环境威胁：**
- 雷暴：高地不定期爆发雷暴。雷暴期间，闪电频率极高，在露天区域行走有概率被击中。被雷电击中不会即死，但会造成灵焰的暂时紊乱（所有能力冷却时间翻倍，持续一段时间）。玩家可以通过观察云层颜色（青白色=强雷暴将至）提前寻找掩体。
- 断崖与坠石：裂谷边缘的岩层不稳定。在裂谷边缘冲刺有概率触发岩层崩塌。掉落深谷通常意味着死亡（除非装备了滑翔披风或攀爬钩爪）。
- 静电灵痕：高地上空的电荷在灵质层面形成了一种独特的现象——"静电灵痕"。这些微小的电弧会在空气中随机闪现，它们对蚀者有强吸引力。在裂霆高地，蚀者出现的频率高于其他区域，且更倾向于在雷暴期间活动。

**可发现的秘密：**
- 天穹之冠的圆顶密室：泽法利昂主殿的核心区域。进入需要依次激活散布在高地各处的三座"雷针塔"——每座塔需要用灵焰触发其顶部的雷石（触发后会引来一次小型雷暴——考验准备是否充分）。圆顶密室中保存着泽法利昂最后的雷霆核心碎片——拾取后可解锁裂霆系灵格能力"天霆余响"（造成伤害时有概率触发一次小范围雷击）。
- 泽法利昂的焦痕地图：散布在高地各处的焦痕中，有七道焦痕的形状恰好拼合成一张完整的地图——指向一处隐藏的"风暴之锤"锻造坊遗址（托尔迦曾在此维护他的风暴之锤）。遗址中有稀有锻造材料和一张"雷铸配方"。
- 无光之渊的渊息孔：高地某处有一个直径约三米的黑色洞口——哈德昂旧领地的一个渊息孔。站在孔边可以听到从无光之渊中传出的死者的低语。玩家可以通过"倾听"获得随机信息——可能是某个已死角色的遗言片段，可能是一个未完成任务的线索，也可能只是一声叹息。

**灵植与资源：**
- 雷击木（普通）：被雷击后枯死的圣橡残枝木块，木质中封存了雷电轰击瞬间的灵质脉冲。是制作灵焰弩弩箭和陷阱触发器的核心材料。
- 电弧花（稀有）：只生长在经常被雷击的岩石表面的紫色小花。花瓣在雷暴期间会发出微弱的电弧。可用于制作"雷抗药剂"（临时抵抗雷暴伤害）和"电弧陷阱"。
- 天铁矿石：银色金属矿石，只存在于被雷击过的岩层中——这种金属"记得"泽法利昂的灵焰频率，所以具有天然的灵质排异性。锻造灵质护甲的核心材料。
- 雷心水晶（极稀有）：雷暴期间，偶尔一道特别强烈的闪电击中天穹之冠的圆顶后，圆顶上会凝结出一颗不断发出噼啪声的蓝色水晶。仅在雷暴天气中出现，每个雷暴周期最多产生一颗。

**区域NPC：**
- 天霆朝圣者赛拉：一位从暮色聚落出发、试图徒步到达天穹之冠圆顶的年轻灵织者。她相信在圆顶中祈祷能听到泽法利昂对她说的一句话。玩家可以帮助她安全穿越裂霆高地，沿途保护她免受蚀者和雷暴伤害。到达圆顶后，她确实听到了什么——内容她不透露，但她的灵焰颜色从此变成了淡银色。奖励是"天霆信标"——一件可放置的信标，在3D竞技模式中可召唤一次小型雷击。
- 雷铸者科尔姆：一个驻扎在天穹之冠山脚铁匠铺的老锻甲灵织者。他自称知道如何修复天穹之冠圆顶上的一处破损——但他需要的材料散落在裂霆高地的四个角落。他会在玩家帮他收集材料后，用天铁为玩家锻造一件独一无二的装备。任务链最终解锁"雷铸锻炉"——允许玩家在自己的营地中建造一座小型天铁锻造站。

### 1.3 幽翠残林（翠庭圣座遗址）

**地貌特征：** 曾经繁茂的圣林如今由无数银灰色枯死圣橡构成。树干仍然站立，但内部已经空了——它们的灵质层面部分被编织进了帷幕。地面覆盖着厚厚的落叶层（落叶永远不会完全腐烂——它们只是从棕色变为银白色）。林间偶尔可以看到圣橡根系在灵质层面留下的"根语印痕"——如透明丝线般在空气中短暂闪烁。

**环境威胁：**
- 迷雾：林地中常年弥漫着一种特殊的雾——不是水雾，是灵质雾。迷雾会干扰灵焰感知（感知范围减少），同时让方向感变得不可靠。深入森林过远而没有标记路径的话容易迷路。玩家需要沿途在圣橡树干上刻下"灵焰标记"来记录路线。
- 根语错觉：残存的圣橡根系网络虽然已经无法传递完整的思维，但仍然会释放出碎片的"根语回响"。这些回响会以幻觉的形式出现在玩家面前——可能是一段记忆的重演，可能是一个声音的呼唤，也可能是诱导玩家偏离安全路线的陷阱。长时间暴露在根语回响中，玩家的灵焰会开始"共情"——无意识地吸收根语中的悲伤情绪，导致灵焰颜色暂时变暗（不影响战斗能力，但影响NPC互动——NPC会更不信任"灵焰暗淡"的人）。
- 渡鸦群：莫丽甘娜的渡鸦后代仍然栖息在这片林地中。它们通常不攻击人类，但在玩家触碰渡鸦冢时会发动集体攻击。被渡鸦群攻击不会造成灵焰伤害，但会强制驱离玩家离开该区域。

**可发现的秘密：**
- 莫丽甘娜的渡鸦冢：位于林地深处，由无数渡鸦风干后的尸体堆积成一座小山——山的轮廓精确地呈一只俯冲的渡鸦形状。在血月之夜，冢上会发出暗紫色的微光。与渡鸦冢交互可以触发莫丽甘娜留下的最后一段记忆——一段由根语保存的全息影像，展示她走进渡鸦群前的最后一刻。解锁这段记忆后，玩家获得"渡鸦之眼"灵格能力（在探索中可短暂以渡鸦视角俯视周围区域）。
- 圣橡母树的最后幼苗：达格德里安在献祭前从他的掌心长出的那棵圣橡幼苗——据说被一个灵织者带回了圣林并种在了一个秘密地点。这棵幼苗至今仍在生长，虽然极其缓慢。找到它的玩家可以用自己的灵焰浇灌它（消耗自身灵焰储量），换取一次"根语全知"——短暂看到整片林地所有NPC和怪物的位置。
- 根语地下网络：在几棵特定圣橡的根部，可以发现进入"根语地下网络"的入口——一个由圣橡根系编织成的、在灵质层面可见的地下通道系统。这个网络连接着林地中多处隐秘地点，可以用来快速穿越林地或找到隐藏的资源点。但在地下网络中停留会持续消耗灵焰（因为网络本身是灵质维度的——物理身体在其中不稳定）。

**灵植与资源：**
- 圣橡落叶（普通）：从枯死圣橡的根部收集。落叶中仍含有微弱的灵质内容——是愈灵汤剂的核心原料。在灵质层面加热后可以释放出短暂的"根语回音"（可用于迷惑附近的蚀者）。
- 幽翠苔藓（稀有）：只生长在最古老的圣橡树干北侧的翠绿色苔藓。含有高浓度的灵质恢复成分。可制作"愈灵药膏"（直接涂抹在灵焰裂缝上以加速愈合）。
- 琥珀灵脂（稀有）：圣橡树干裂缝中渗出的金黄色树脂——是圣橡在被编入帷幕前的最后一段时间中，将多余灵质转化为物理形态储存的产物。燃烧后可产生大量灵质暖意，用于大型营地篝火和愈灵仪式。
- 根语核心（极稀有）：在地下根语网络的某个深处节点，可以找到一颗仍在脉动的木质球体——它是根语网络最后残存的中枢。拾取后可以在竞技模式的3D地图中解锁"根语感知"能力——短暂获知全图所有玩家的灵焰状态（但每次使用后本局能力全部进入冷却）。

**区域NPC：**
- 守林人奥克塔维亚：一位在圣林边缘住了三十年的老妇人，自称是达格德里安在献祭前见过的最后一个凡人。她能"听到"圣橡在哭泣（通过根语的极微弱残留），并试图用她自制的愈灵汤剂浇灌枯死的圣橡——虽然她知道这不会有任何效果。她向玩家提供幽翠残林的完整地图和所有安全路线的标记。她的任务线围绕"为每一棵她还记得名字的圣橡浇灌一次汤剂"展开。完成全部浇灌后，林地中会出现一棵重新长出嫩叶的圣橡——这是达格德里安的无声回应。
- 渡鸦守护者柯文：一个不说话的少年，住在渡鸦冢附近的一间由枯枝搭建的简陋小屋里。他能与渡鸦群沟通（通过一种连他自己也无法解释的、与莫丽甘娜灵焰残响之间的微弱共鸣）。他可以训练一只渡鸦作为玩家的同伴（在探索中自动标记附近的资源和怪物位置）。任务线最终解锁"渡鸦伙伴"——一艘可在开放世界使用的飞行渡鸦，可携带少量物品、帮助侦察、或传送信件给其他在线玩家。

### 1.4 骨灰平原（金沙圣座遗址）

**地貌特征：** 一望无际的灰白色平原，地面覆盖着细密的粉末状物质——不是沙，是亿万个被遗忘的名字风化后剩下的骨灰。平原上零星分布着破碎的白石板碎片——伊希丝命符殿中散落出来的灵符记录碎片。地平线上偶尔可以看到沙尘卷起的微型旋风——那是一个名字在离开。

**环境威胁：**
- 灵焰缓慢流失：骨灰平原是所有区域中最安静也最危险的——它不会主动攻击你，但会持续缓慢地吸附你灵焰中的热量。待得越久，灵焰越薄。每一天的流失率是大约万分之五——看似微不足道，但如果你深入平原腹地需要走三天，总流失量就是致命的。玩家必须携带足够的愈灵汤剂或"灵焰暖石"来补充，并且必须在灵焰储量降至危险线之前找到可扎营的安全点。
- 灵符失读：当你长时间凝视白石板碎片上的灵符时，伊希丝的编码系统会在灵质层面触发一种"信息过载"效应——你的灵焰会开始自动尝试解码灵符，消耗大量精神集中力。连续阅读超过十块碎片而未休息，会陷入"灵符痴迷"状态——视野模糊、反应迟钝、无法感知蚀者靠近。
- 名字的回声：有时，一口即将离开的名字会在你耳边低语它最后记得的东西——可能是一个地点、一个人名、或一段断掉的誓言。大多数回声是无害的——但有些名字属于被背叛而死的人。那些名字的回声携带着一段强烈的情绪残留，会导致听到的人短暂陷入那段情绪的共情中（恐惧、愤怒、绝望）。

**可发现的秘密：**

### 1.5 白石帝国遗址（律法圣座遗址）

**地貌特征：** 白色大理石建筑群的大面积废墟——朱沃纳斯万律之城的遗骸。虽然大部分建筑已经崩塌，但万律石的材质使废墟保持着一种奇异的"有序感"——碎石似乎按照某种看不见的规则排列。最显著的地标是律法广场——一片巨大的石面广场，朱沃纳斯最后一次演讲的脚印仍清晰可见。广场周围散落着倒塌的柱廊、半埋的雕像、和那些曾经用来讨论万域通则条款的辩论厅残骸。

**环境威胁：**

- 塌方与不稳定的拱门：废墟中的某些结构看起来稳定但实际上随时可能坍塌。经过时需要观察地面上的裂缝（塌方前兆）或听取石头发出的细微断裂声。
- 守律石像：白石帝国的某些走廊和入口处仍然站立着半毁的"守律石像"——朱沃纳斯时代的自动守护装置。大部分已经失效，但有些仍在运作——它们不会主动攻击，但会拦截"没有权限"的人通过特定区域。通过守律石像需要找到对应的"律法通行符"——散布在废墟中的一次性格印石。

**可发现的秘密：**
- 朱沃纳斯的隐藏之字：万域通则最后一页最后一行的那个被故意留白的字——据说在白石帝国遗址的某处有一个对应的物理版本。它不是写在纸上——是刻在一扇木门上（那扇门不在任何已知的入口处——它独立存在，悬浮在废墟深处的某一堵残墙中）。找到并开启这扇门的玩家，会在门后看到一个他们此生以为不可能再见到的人——具体是谁取决于该玩家的角色背景故事。这次相遇不是真的（是朱沃纳斯借用雅努维翁之门的一部分力量留下的"可能的相遇"），但它在灵焰层面留下的影响是真的——全属性短暂大幅提升。

- 辩论厅的回声：律法广场旁的一个半塌的辩论厅中，保存着朱沃纳斯主持的最后一场辩论的"回声"——万律石的灵质记忆将当时在场所有人的发言全部记录了下来。站在辩论厅中央特定位置，可以听到那段辩论的完整内容——其中包含朱沃纳斯对千面之影的预言性警告（他称其为"通则之外的条款"）。听完整个回声后，玩家对蚀痕的抵抗力永久小幅提升。

## 二、生存系统：灵焰管理

### 2.1 灵焰的核心机制

灵焰是玩家在开放世界中的生命本质——它不是传统意义上的"生命值"，而是一个多维度状态系统。

**灵焰属性：**
- 灵焰储量（Lumen Reserve）：灵焰当前的"亮度"，相当于生命值。受食物、庇护、愈灵、和蚀痕侵蚀影响。储量降至零即死亡（灵焰离体）。
- 灵焰纯度（Lumen Purity）：灵焰被蚀痕污染的程度。纯度为100%表示完全纯净，0%表示即将完全蚀变。纯度会因接触裂隙、被蚀者攻击、或遭受极端精神冲击而下降。低纯度时，角色会出现视觉异常（颜色变得不准确）、NPC信任度降低、且更容易被蚀者发现。
- 灵焰温度（Lumen Temperature）：反映灵焰的活跃程度。温度受环境（寒冷区域持续降温和温暖区域恢复）、食物（热食提升温度）、和庇护（在篝火旁恢复）影响。低温度时移动速度降低、灵焰恢复速度变慢；过高的温度（如在裂霆高地被雷击后）会导致灵焰紊乱。最适宜温度维持在一个中等范围。
- 灵焰完整度（Lumen Integrity）：反映灵焰的结构性健康。被攻击或使用消耗性能力时会产生"灵焰裂缝"。裂缝本身不是蚀痕，但裂缝处更容易被蚀痕附着。裂缝可以通过愈灵汤剂和时间自然恢复。裂缝过多（完整度过低）时，角色更容易被击杀。

**复合式生存循环：**
1. 食物（灵质食物）：进食恢复灵焰温度，并极缓慢地恢复灵焰储量。灵质含量越高的食物（如用灵植烹饪的料理），恢复效果越强。
2. 庇护（篝火/营地/聚落）：在篝火旁休息加速灵焰储量和温度恢复，并可主动进行愈灵操作。完全不动地"静养"状态恢复速度最快。
3. 愈灵（药剂/愈灵师/修行）：使用愈灵汤剂、寻求愈灵师帮助、或修习呼吸法来修复灵焰裂缝、清除蚀痕附着。

### 2.2 食物系统

**食材来源：**
- 采集：在六个区域采集可食用植物（幽翠残林的野菌、永冬山脉的冰苔莓、骨灰平原的平原苔等）。
- 狩猎：猎杀环境中的中立动物（雪兔、石羊、林地鹿）。猎杀会消耗灵焰完整度（因为杀戮行为在灵质层面上会撕开微小裂缝——这是世界观核心设定：暴力本身就在侵蚀使用暴力的人）。
- 灵质料理：将普通食材与灵植（霜星苔、电弧花等）混合烹饪，制作具有特殊效果的料理。灵质料理的效果远超普通食物。

**烹饪系统：**
玩家可以在营地篝火或聚落厨房中烹饪。烹饪需要时间引导，时间越长材料越充分，成品品质越高。不同区域的特殊食材组合可制作专属料理：
- 霜星暖汤（永冬山脉）：霜星苔+雪兔肉+洁净水 → 灵焰温度大幅恢复，且在接下来一段时间内对寒冷环境有抗性。
- 雷霆烤鱼（裂霆高地）：雷击木炭烤石羊肉+电弧花粉 → 灵焰活跃度提升，移动速度暂时增加。
- 根语香茶（幽翠残林）：圣橡落叶+幽翠苔藓+洁净水 → 灵焰纯度和完整度同时恢复。在营火旁饮用效果翻倍。
- 命符粥（骨灰平原）：骨灰平原苔+灵符碎片粉末（极少量）+任意谷物 → 恢复大量灵焰储量。但有极低概率在食后触发一段"名字的回声"（看到一段随机记忆）。
- 石心面包（白石帝国遗址）：律法苔+普通面粉 → 灵焰稳固性提升——受到攻击时产生的裂缝面积减少。
- 清心禅茶（雾隐禅院）：禅院苔+洁净水 → 灵焰波动大幅降低，隐藏自身不被蚀者察觉。

### 2.3 庇护系统

庇护分为三个层级：

**临时篝火（野外）：**
在最基本的野外营地——一堆篝火——旁边，玩家可以恢复灵焰温度和储量（速度慢但持续）。需要燃料（木材或灵植残料）。篝火提供光照（驱散小型环境生物）、可用于烹饪、并为周围小范围提供"灵质暖区"（蚀者不会主动进入暖区，但如果被攻击，暖区不能阻挡）。篝火不可加固，不能在裂隙风暴中保护玩家。

**简易营地（建造系统产物）：**
由玩家在野外建造——包括帐篷（提供上限睡眠和庇护）、篝火升级版（更大的烹饪和愈灵空间）、简易围栏（减缓蚀者靠近速度）。营地中的灵质暖区范围更大，且可以放置"灵焰暖石"来持续补充玩家的灵焰温度。营地可以被裂隙风暴破坏——需要定期维护。

**加固庇护所（建造系统产物——需要与聚落NPC合作）：**
在特定位置（通常靠近聚落或安全地带）由玩家和NPC联手建设的半永久性建筑——包括坚固墙壁、灵质门锁、愈灵躺椅、储物空间、工作台（锻造/烹饪/愈灵）。加固庇护所在裂隙风暴中不会被破坏，且蚀者无法进入（但更高等级的蚀者——如冥僧人——仍然可能找到入口）。每个加固庇护所消耗大量资源，但在该区域提供了一个完全安全的"重生点"——如果玩家在该区域死亡，可以选择在该庇护所重生（而非回到暮色聚落初始点）。

## 三、建造系统

### 3.1 建造概览

玩家在离开暮色聚落探索野外时，可以在任意平坦地面建立营地。建造系统使用模块化组件——玩家收集材料后在建造菜单中选取组件并放置在可建造区域内。所有组件需要时间引导（类似"建造动画"），在引导期间无法移动或战斗。合作模式下，多个玩家可以共同引导同一个组件来缩短建造时间。

### 3.2 营地组件

**基础组件（可用基础材料制作）：**
- 篝火：需要木材x5 + 燧石x1。提供光照、烹饪台、灵质暖区。
- 帐篷：需要布料x8 + 木杆x4。提供睡眠空间（恢复大量灵焰——相当于"存档点"）。帐篷内可放置一个物品箱。
- 简易围栏：需要木材x10。围绕营地的低矮栅栏——蚀者接近时会被短暂减速（约3秒）然后翻过。给玩家提供反应时间。
- 工作台：需要木材x15 + 石材x5。允许在野外进行基础锻造和药剂制作。

**中级组件（需要稀有材料）：**
- 灵质篝火：篝火 + 琥珀灵脂x2 + 灵质暖石x1。暖区范围扩大，且灵焰恢复速度翻倍。燃烧时产生一种微弱的金色光芒，对蚀者有微弱驱散效果。
- 愈灵躺椅：需要木材x12 + 幽翠苔藓x5 + 布料x4。躺在上面时灵焰裂缝修复速度翻倍。可同时容纳一人使用。
- 锻造台：工作台 + 天铁矿石x5 + 万律石碎片x3。允许在野外进行灵质护甲和门锁的锻造。
- 观测塔（简易版）：需要木材x20 + 石材x10 + 灵符碎片x2。搭建一座小型瞭望塔——站在塔顶时视野范围扩大。同时可以用塔顶的灵质感应器监测周围区域的裂隙风暴预警。
- 储物箱（加固版）：需要木材x8 + 铁锭x2。储存的物品不会因营地被破坏而丢失（普通物品箱中的物品在营地被毁后散落在地上，需要在时限内找回）。

**高级组件（需要极稀有材料 + NPC合作）：**
- 灵质门锁：锻甲灵织者锻造。安装在营地入口（如果有围栏或墙壁）。蚀者完全无法进入营地。但门锁有耐久度——每次被蚀者攻击会消耗耐久，归零后失效。需要锻甲灵织者定期维护。
- 愈灵室：愈灵躺椅 + 愈灵汤剂库存 + 石材x20。一个小型封闭空间。在其中进行愈灵时，所有愈灵效果翻倍。同时愈灵室本身有微弱的蚀痕净化效果——在室中待一段时间，灵焰中附着的蚀痕会缓慢脱落。
- 雷铸锻造站：完成"雷铸者科尔姆"任务链后解锁。在营地中建造的专用锻造设施——可锻造使用天铁和天霆能量的特殊装备和武器。
- 蓄水池：在营地中有水源的条件下建造。存储洁净水供烹饪和药剂制作使用。同时蓄水池中的水可以被"灵质化"——加入圣橡落叶浸泡后产生灵质水（用于更高级的药剂配方）。

### 3.3 营地选址策略

不同区域适合不同位置的营地：
- 永冬山脉：优先选择靠近温泉的天然洞穴——保温、有水源、且天然保护免受暴风雪。缺点是可能已被冰封蚀者占据，需要先清理。
- 裂霆高地：优先选择峡谷中的天然岩棚——提供防雷避护。但峡谷底部的灵质信号较差，营地中的灵质篝火范围会缩小。
- 幽翠残林：优先选择圣橡树洞——枯死圣橡的巨大树干内部是天然庇护所。且树洞中的根语回响更温和，有助于愈灵。缺点是在迷雾中很难找到回树洞的路。
- 骨灰平原：几乎没有任何天然庇护所。玩家必须完全依靠自己建造的营地。建议在平原边缘扎营——深入腹地前确保有足够的愈灵汤剂和灵质暖石。
- 白石帝国遗址：优先选择半塌但仍能遮风避雨的万律石建筑内部。万律石对蚀者有微弱的天然排斥——住在万律石建筑中相当于额外一层被动防护。
- 雾隐禅院：岛上空间极有限——只能在禅房旁搭建非常小型的临时篝火。但此处不需要防御——禅院是蚀者唯一无法进入的区域（玉穹尊者残留意境使其"不可见"）。

## 四、灵质生态

### 4.1 灵植采集

灵植是开放世界中最重要的可再生资源之一。灵植的分布、生长周期、采集难度因区域和植物种类而异。

**采集机制：**
- 灵焰感知定位：大部分灵植在普通视野中与普通植物无异。玩家需要开启灵焰感知（消耗少量灵焰）来看到它们发出的微弱灵质光芒。不同灵植的光芒颜色不同——霜星苔是淡青色、电弧花是紫色、幽翠苔藓是翠绿色。
- 采集工具：基础灵植可以用手采集。稀有和极稀有灵植需要专用工具——如冰凿（用于永冬山脉的冰层中的灵植）、绝缘手套（用于电弧花）、灵质刀（用于圣橡树脂）。工具的等级影响采集成功率和获得的材料数量。
- 可持续采集：大部分灵植采集后会在一段时间（游戏内数日到数周）后再生。如果一次性采光一片区域的所有灵植，该区域会进入"枯竭"状态——再生时间翻倍。灵植采集规则鼓励玩家分散采集而非在一个点上反复采。

### 4.2 灵质矿物开采

灵质矿物是锻造灵质装备、建造高级营地组件、制作特殊道具的核心材料。

**开采机制：**
- 矿脉探查：与灵植不同，灵质矿脉埋在地表以下。玩家需要使用"灵质探测锤"（可在聚落铁匠铺制作）来探查脚下土壤中是否有矿脉信号。探测锤会释放一次灵质脉冲——如果下方有矿脉，脉冲会反弹回一个特殊频率的信号（不同的矿物有不同的信号）。
- 开采工具：基础矿石用铁镐。灵质矿石（永冻铁、天铁矿、星陨铁）需要用灵质镐——由铁镐注入愈灵师的灵焰祝福后升级而来。
- 矿脉枯竭与再生：每条矿脉有最大开采次数（如可开采15-25次）。采尽后矿脉永久消失，但在该区域的另一个位置会随机生成一条新矿脉（模拟地质活动）。这意味着矿脉位置不是固定的——需要持续探查。

### 4.3 古代遗物发掘

古代遗物是诸神时代留下的、不可再生的稀有物品。它们通常具有独特的功能或可以换取大量资源。

**发掘机制：**
- 线索碎片的收集：在古代遗物可能存在的区域，玩家可以通过以下方式获取线索——阅读灵符碎片、倾听渊息孔的低语、与特定NPC交谈、解读根语回响、或解析帷幕低语中的信息。线索会在大地图上逐渐缩小遗物的可能位置范围。
- 发掘过程：确定位置后，玩家需要用灵质探测锤确认精确的发掘点，然后进行挖掘（有引导时间——数秒到数分钟，取决于遗物的类型和深度）。在挖掘过程中可能会触发陷阱（远古遗留的防护机制——需要灵质反应来解除）或被附近的蚀者发现（挖掘释放的灵质波动会吸引蚀者）。
- 一次性的发现：每个遗物在世界中是唯一的——一个玩家发现并拾取后，该遗物不再生成。部分遗物可以在合作模式中共享（所有参与发掘的玩家都获得一份复制品），但大多数只能由一人获得。

**遗物类型：**
- 灵格遗物：永久提升角色属性或解锁新能力（如"金瞳水晶"）。
- 知识遗物：解锁世界观秘密、新配方、或隐藏任务线（如"瓦尔德林的霜纹石板"）。
- 货币遗物：可以在暮色聚落中出售换取大量资源或独特物品的高价值遗物。
- 竞技遗物：可以在竞技模式中使用的特殊道具或外观（如"天霆信标"）。

## 五、裂隙风暴（随机世界事件）

### 5.1 机制概述

裂隙风暴是帷幕暂时变薄、裂隙力量增强的区域性灾难事件。它不是在特定时间触发的——而是当一个区域的"裂隙压力"累积到临界值时自动触发。裂隙压力受多个因素影响：该区域近期死亡人数（包括蚀者死亡）、该区域玩家灵质活动的总量、千面之影的全局活动周期、以及（在极少情况下）特定玩家的"不道德选择"（如反复在同一区域出卖NPC或抢夺他人资源）。

### 5.2 风暴阶段

**第一阶段——裂隙预警（持续约3-5分钟实际时间）：**
- 天空颜色变暗、月光变成深红色。
- 所有玩家屏幕边缘出现微弱的黑色纹路（帷幕变薄的视觉表现）。
- 区域中的所有灵植发出的灵质光芒开始不稳定地闪烁。
- 灵织者和帷幕学者会收到特殊的"裂隙低语"（系统提示风暴即将来临的大致时间和位置）。
- 玩家在此阶段应尽快找到庇护所或向营地撤退。

**第二阶段——裂隙撕裂（持续约10-15分钟）：**
- 天空中出现可见的黑色裂隙——帷幕真的被撕裂了微小的缝隙。
- 裂隙碎片（小型黑色飘浮物）从裂隙中飘出。接触碎片会直接造成灵焰纯度下降和蚀痕附着。
- 区域中的所有蚀者获得增益——移动速度增加、攻击力增强、且对灵质暖区不再完全回避。
- 野外环境变得危险——灵植暂时被"感染"（采集会带来蚀痕附着风险）、矿脉产出减半、营地中的篝火范围缩小。
- 在风暴期间死亡，灵焰不会正常离体后被帷幕接收——而是被裂隙捞走（该角色的灵焰对该玩家暂时不可用——需要通过愈灵仪式在聚落中"召回"）。

**第三阶段——裂隙消退（持续约2分钟）：**
- 裂隙逐渐闭合，天空颜色恢复。
- 风暴期间出现的裂隙碎片开始消散——但消散前最后一次闪烁会掉落"裂隙残渣"（一种风暴后独有的可收集物，是极为罕见的制作材料）。
- 区域中的所有蚀者重新隐藏——但它们在风暴后一周内（游戏内时间）更加活跃。
- 灵植在风暴后会进入一个短暂的"高产期"（约一游戏日）——它们的灵质光芒更强，采集收益加倍（可能是风暴中的灵质辐射刺激了它们生长）。

### 5.3 裂隙风暴生存策略

- 提前准备：在每个区域的营地中储备愈灵汤剂、灵质暖石、和足够的食物。营地中的观测塔（简易版）可以提前探测到裂隙压力上升。
- 固守营地：如果营地有加固庇护所级别——躲在其中，点上灵质篝火等待风暴过去。不要在风暴期间外出探索。
- 如果没有营地：寻找区域中的天然庇护点——永冬山脉的冰洞、裂霆高地的岩棚、幽翠残林的圣橡树洞、白石帝国遗址的半塌建筑。天然庇护点不能完全阻挡裂隙碎片，但能减少碎片接触概率。
- 合作防御：合作模式中，多个玩家可以轮流维持一面"灵焰护盾"（需要灵质材料消耗）来保护整个队伍。一个人维持护盾时其他人可以趁机愈灵或烹饪。
- 禁忌：在裂隙风暴期间不要进行愈灵——愈灵时的灵焰外放会吸引裂隙碎片。不要在风暴期间靠近渊息孔——渊息孔在风暴中会变成"裂隙喷口"（大量裂隙碎片从孔中涌出）。

### 5.4 风暴后的机会

裂隙风暴虽然危险，但也是收集稀有资源的最佳时机：
- 裂隙残渣（风暴后散落在地上）：唯一可用于制作"裂隙抗性护甲"的核心材料。
- 灵植高产期：风暴后短时间内所有灵植采集产出加倍。
- 蚀者掉落：风暴期间击杀的蚀者会掉落他们平时不掉落的"蚀痕核心"——可以用于研究蚀痕机制或制作特殊道具的材料。

### 6.1 永冬山脉——"等待之人"

**触发：** 在永冬山脉深处发现霜巨灵冰封群像后，与群像交互触发。

**奖励：** "霜巨灵的祝福"——灵焰最大储量永久+20%。永冬山脉新增一头友善霜巨灵NPC（可用资源交换冰晶灵髓）。在竞技模式中解锁"霜巨人形态"外观（角色体型短暂变大，仅外观，不影响战斗）。

### 6.2 裂霆高地——"雷鸣的馈赠"

**触发：** 在裂霆高地帮助天霆朝圣者赛拉安全到达天穹之冠圆顶后触发。

**任务流程：**
1. 赛拉在圆顶中听到了泽法利昂对她说的六个字（她未透露内容）。但她告诉你，圆顶中的"雷声"不只是泽法利昂一个人的声音——它是由裂霆圣座所有已死诸神的最后一段灵焰残响重叠而成的。
2. 你需要收集裂霆三兄弟各自的残响——哈德昂的（在渊息孔中倾听——需要找到高地中特定一个"深到听不到底"的渊息孔）、波里冬的（需要潜入沉没之殿上方水域——在灵质层面接收水中的低频振动）、泽法利昂的（雷暴期间在天穹之冠圆顶中央站到被雷击中——被击中后残响自动刻入灵焰）。

5. 听到那个音节后，玩家获得一次"三兄弟的对话"——三兄弟在献祭前的最后一次完整的、面对面的对话内容——以一段根语式的全息记忆形式在共鸣石上播放。内容涉及他们对千面之影的早期应对策略和他们留在帷幕各处的三个"秘密遗产"的位置。

**奖励：** 三兄弟遗产的位置坐标（涉及三个新的隐藏发掘点——各含一件极稀有装备）。"雷鸣的馈赠"灵格——释放灵质能力时有概率追加一次小型雷击。

### 6.4 骨灰平原——"她最后一个名字"

**触发：** 在骨灰平原完成拾忆者玛拉的任务（帮她收集未记录的白石板碎片）后触发。

**奖励：** "被救赎的名字"——一本新的羊皮纸册子（饰品，放在营地中时，营地附近的所有灵质恢复速度+10%）。"伊希丝的秘密"灵格——阅读灵符碎片时不再触发"灵符痴迷"效果。在竞技模式中解锁"命符师"外观（角色手臂上出现微弱的金色灵符纹路）。

## 七、角色养成：灵格系统

### 7.1 灵格概述

灵格是玩家在开放世界模式中的等级和养成系统。灵格等级决定了一系列属性——灵焰最大储量、灵质感知范围、愈灵效率、建造速度——以及可装备的"灵格能力"数量。

**灵格经验来源：**
- 区域探索：每解锁一个新的地图区域或发现一个隐藏地点获得经验。
- 任务完成：每个任务（主线/支线/NPC委托）获得经验。
- 资源采集：灵植采集和矿物开采获得微量经验。
- 古代遗物发掘：每次完成发掘获得大量经验。
- 裂隙风暴存活：在裂隙风暴中存活（不掉到灵焰极限以下）获得经验。
- 合作互助：在合作模式中帮助队友（愈灵、共享资源、共同建造）获得额外经验。

### 7.2 灵格等级与能力解锁

**灵格阶段一——"灵焰初醒"（等级1-10）：**
- 解锁基础灵焰感知能力。
- 解锁基础愈灵操作（可对自己使用初级愈灵汤剂）。
- 解锁基础建造（篝火和帐篷）。
- 解锁一个灵格能力槽位。

**灵格阶段二——"灵焰掌控"（等级11-25）：**
- 灵质感知范围+30%。
- 解锁中级建造组件（灵质篝火、愈灵躺椅、锻造台）。
- 解锁第二个灵格能力槽位。
- 可装备区域专属灵格能力（如"雷鸣的馈赠"——从裂霆高地任务中获得）。

**灵格阶段三——"灵焰精通"（等级26-50）：**
- 灵焰裂缝自然恢复速度翻倍。
- 解锁高级建造组件（愈灵室、观测塔简易版）。
- 解锁第三个灵格能力槽位。
- 裂隙风暴预警时间延长（给更多准备时间）。

**灵格阶段四——"帷幕感知"（等级51-80）：**
- 帷幕低语解读成功率大幅提升（获得更多和更准确的探索线索）。
- 解锁极稀有材料的使用权限（冰晶灵髓、雷心水晶等可被直接用于制作）。
- 解锁第四个灵格能力槽位。
- 在裂隙风暴期间不会再被裂隙碎片"即死"——改为重伤（灵焰储量降到1）。

**灵格阶段五——"灵焰大师"（等级81-100）：**
- 全区域环境威胁减免30%。
- 愈灵效率翻倍。
- 解锁全部建造组件。
- 解锁第五个灵格能力槽位。
- 可建造"灵焰共鸣器"——一个允许在营地中与其他玩家共享灵格被动效果的设备（合作模式中）。

### 7.3 灵格能力的分类

**被动能力（每个灵格槽位可装备一个被动能力）：**
- 区域类：在特定区域获得额外抗性或收益（如"独目者的远见"——永冬山脉视野+10%）。
- 生产类：采集/开采/建造效率提升。
- 生存类：灵焰属性加成或消耗减免。
- 社交类：NPC信任度提升、交易折扣。

**主动能力（从灵格里层身份或遗物中获得）：**
- 战斗类：区域专属攻击能力（如"天霆余响"——造成雷击追加伤害）。
- 探索类：环境穿越能力（如"渡鸦之眼"——俯视侦察）。
- 愈灵类：在限定条件内愈灵效率大幅提升。

灵格能力在竞技模式中不能直接使用——但解锁灵格能力会在竞技模式的"灵格加成"栏中提供对应的外观和微量属性加成（见第八节）。

## 八、与竞技模式的联动

### 8.1 资源双向流通

开放世界探索模式产出的资源可通过以下方式带入竞技模式（3D追逃/桌游模式）：

**外观解锁：**
- 特殊资源（金瞳水晶、霜纹石板、意境玉简等遗物）在开放世界中拾取后，可在竞技模式中解锁对应的角色外观。例如：
  - 金瞳水晶 -> "守望者之眼"外观（角色眼睛呈现金色灵光效果）
  - 霜纹石板 -> "独目者的远见"外观（角色左眼环绕淡青色符文）
  - 渡鸦冢羽毛 -> "渡鸦守护"外观（角色肩膀停驻一只渡鸦——纯外观）
  - 永燃蜡烛残蜡 -> "引路之光"外观（角色周围有微弱的暖色光环）
  - 雾霭玉 -> "雾隐行者"外观（角色移动时有淡淡的白雾尾迹）

**灵格加成：**
- 玩家在开放世界模式中达到的灵格等级会为竞技模式中的角色提供微量属性加成。加成的幅度被严格限制为了防止破坏竞技公平性——每10级灵格提供约1-2%的属性提升，且会在匹配中列入考量（灵格等级相近的玩家匹配到同一局）。
- 加成类型：灵焰初始储量微增、灵质感知半径微增、愈灵(如果竞技模式中有此机制)效率微增。
- 重要限制：灵格加成在排位赛中完全禁用——确保排位赛的完全公平性。仅休闲匹配中启用。

**竞技专属道具：**
- 开放世界中制作的部分消耗品可以带入竞技模式作为"初始携带物"（每局可选择携带一件）：
  - 愈灵汤剂（小）：使用后恢复少量灵焰（仅限3D追逃模式）。
  - 灵质暖石：在竞技模式的雾天/暴风雪天气中减少视野惩罚。
  - 灵焰弩箭（一支）：在追逃模式中可进行一次远程灵质攻击。
  - 清心香：使用后短时间内灵焰波动减小——狼人的灵质感知更难发现你。
- 携带物是消耗品——每次使用后需要重新在开放世界中制作补充。
- 在桌游模式中，携带物以"一次性能力卡"的形式体现（在白天讨论阶段使用）。

### 8.2 叙事联动

**全局叙事进度：**
- 开放世界模式中的关键任务完成会影响竞技模式中的"世界状态"：
  - 完成"等待之人"任务 -> 竞技模式的永冬山脉主题地图中，背景中可以看到苏醒的霜巨灵在冰原上行走。
  - 完成"为她种一棵树" -> 幽翠残林主题地图中，迷雾密度轻微减少。
  - 完成"茶与雾之间" -> 雾隐禅院主题地图中，石灯笼的光更温暖。
- 这些变化是纯视觉/氛围层面的——不影响竞技公平性。

**千面之影的元叙事：**
- 开放世界模式的探索进度累积到一定程度时，竞技模式中的"深渊结局"触发条件会被部分揭示（在游戏中的"指引"低语中给出更明确的警告）。
- 开放世界中发现的关于千面之影的信息会逐渐填充竞技模式中的"真相档案"——一个供所有玩家共同推进的全局收集系统。当服务器上所有玩家收集的真相碎片达到一定数量时，整个玩家社区的千面之影对抗进度会推进——可能解锁新的竞技模式地图、新的角色、或新的世界事件。

### 8.3 两种模式的互补关系

- 竞技模式提供快速、高强度的PvP体验（15-30分钟一局）。
- 开放世界模式提供缓慢、沉浸的PvE体验（以小时为单位的探索循环）。
- 竞技玩家可以通过开放世界模式获得外观和微量加成——增加了长期投入的动机。
- 开放世界玩家可以在竞技模式中使用辛苦收集制作的道具——让探索成果有"秀"的舞台。
- 两种模式共享同一个世界观、同一批角色、同一个叙事进度——但它们从不同维度呈现帷幕之地的故事。

## 九、合作模式特色

### 9.1 联机机制

- 开放世界模式默认为单人模式，但玩家可以邀请好友进入自己的世界（最多4人合作）。
- 合作中的资源分配：灵植和矿石的采集产出按人数倍增（每个人获得同等数量的基础材料）。但古代遗物——作为世界唯一的物品——只有房主可以获得；参与合作的访客获得一份"遗物记忆"（在访客的灵格中记录这个发现，在访客回到自己的世界后，可以在同一位置找到同一件遗物）。
- 建造加速：多人共同引导建造组件时，建造时间按人数非线性缩减（2人=60%时间，3人=40%时间，4人=25%时间）。

### 9.2 合作专属机制

**灵焰共鸣：**
当两个或更多玩家在近距离（同一营地范围内）时，他们的灵焰之间会产生微弱共鸣。共鸣提供：灵焰自然恢复速度微增、愈灵效果微增。共鸣强度随灵焰频率的相似度而变化——同一灵格里层身份的两个玩家之间共鸣更强。

**任务协作：**
某些NPC任务线在合作模式中有额外选项——不同的玩家可以选择不同的对话选项，导致不同的任务分支。比如在"等待之人"任务中，一人与洛根交涉、另一人同时追逐灵质冰原狼——将效率最大化。

**裂隙风暴中的团队分工：**
在风暴期间——一人维持灵焰护盾、一人愈灵、一人守护营地入口——是推荐的分工策略。独行玩家在风暴中更易死亡。

## 十、玩家体验弧线

### 第一阶段：初入帷幕（新手引导，约1-2小时）

玩家从暮色聚落醒来——由奈莉娅(或虚舟)作为一名老练的帷幕学者做新手引导。学习基础的灵焰感知、第一批愈灵汤剂的制作、第一个简易营地的搭建。在暮色聚落周围的安全地带完成几个小型采集任务。

### 第二阶段：向外踏出第一步（单区域初探，约5-10小时）

玩家选择就近的一个区域（推荐从幽翠残林或白石帝国遗址开始——它们的环境威胁相对温和）进行首次深度探索。在这个过程中学习应对环境威胁、对抗蚀者、发掘第一个古代遗物。

### 第三阶段：六域穿行（跨区域探索，约20-40小时）

玩家在所有六个区域间穿行——完成各区域的主线任务、在关键位置建立加固庇护所、收集灵格经验以解锁更多能力和建造组件。这个阶段的玩家逐渐形成自己的探索风格——喜欢独行还是合作、偏好哪个区域、擅长哪种生存策略。

### 第四阶段：真相追寻（深度发掘，约50-80小时）

玩家将六个区域中分散的线索拼合——关于千面之影的起源、关于诸神黄昏的真实性质、关于帷幕还能维持多久。完成区域主线任务、集齐极稀有遗物、解锁大部分灵格能力。

### 第五阶段：永恒守望（终局循环）

在主要任务完成后，裂隙风暴的频率会逐渐增加（千面之影在意识到玩家的探索进度后加大了压力）。终局变成一场持续的生存和维护循环——维护各区域的营地、帮助NPC、在裂隙风暴中保护关键地点、以及在竞技模式中使用积累的资源。没有"通关"——因为帷幕永远需要守护。
