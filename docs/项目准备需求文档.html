一、项目概述

本项目当前为一个完整的在线狼人杀游戏（8种角色、WebSocket实时通信、React前端、Electron桌面端、Android移动端）。计划分三步进化为具有神话叙事背景、双重身份系统、3D非对称竞技玩法的大型游戏。

三步路线：
第一步：角色行为逻辑全面升级，使每个角色都具备猎人级别的策略深度，村民差异化命名
第二步：融合多国神话构建世界观背景，引入"表层身份+里层身份"双重身份系统
第三步：3D自由玩法，类似《黎明杀机》的非对称竞技体验

三、背景故事写作需求

3.1 总体世界观

要素内容要求
世界观名称《帷幕之地》(The Veil-Land)
核心概念诸神黄昏后世界破碎，幸存人类聚集在被"帷幕"包围的土地上——既是避难所也是牢笼。古老血脉在人群中苏醒，每夜狼人肆虐。
神话来源北欧神话（主线）、凯尔特神话、希腊神话、埃及神话、中国民间传说（支线）
基调暗黑北欧童话 — 阴郁但不绝望，有诗意但不做作
篇幅世界观概述 3000字 + 每角色背景 500-800字 × 10 = 总计约10,000字

3.3 需要写作的文本类型

  • 世界观篇章：帷幕之地的由来、诸神黄昏的结局、村庄的形成
  • 角色传记：每人500-800字第一人称或第三人称背景
  • 角色台词：每人3-5句游戏内语音/文字台词（选人时、遇袭时、胜利时、失败时）
  • 物品描述：关键道具的文字（猎枪、毒药、帷幕碎片等）
  • 夜间指引文本：每晚的"指引"文本变体（避免重复感）
  • 结局文本：好/坏/第三方阵营的多种结局描述

五、开发环境搭建指南

5.1 当前阶段（第一步）环境

已有环境，只需确认：
  1. Node.js ≥ 18（node -v 确认）
  2. Git 已安装（git --version 确认）
  3. VS Code + 推荐扩展：ES7+ React Snippets, Prettier, ESLint, Markdown Preview Mermaid
  4. 项目依赖已安装：cd werewolf-online && npm run install:all
  5. 本地开发：npm run dev 同时启动前后端

5.2 第二步新增环境

新增依赖：
  1. 图片处理：安装 Photoshop 或 GIMP（免费替代）
  2. UI设计：注册 Figma 免费账号
  3. AI辅助：根据需要订阅 Midjourney 或使用免费 Stable Diffusion
  4. 音频：安装 Audacity 用于音效编辑
  5. 数据库（可选）：安装 SQLite 或 PostgreSQL 替代 JSON 文件存储

5.3 第三步 — 3D引擎环境

推荐 Unity 方案环境搭建：
  1. 下载 Unity Hub（unity.com/download）
  2. 通过 Unity Hub 安装 Unity 2022 LTS（或更新版）
  3. 安装模块：Windows Build Support + Android Build Support + WebGL Build Support
  4. 下载 Blender 4.x（blender.org）用于3D资产管理
  5. 安装 Visual Studio 2022 Community（Unity的C# IDE）
  6. 注册 Unity Asset Store 账号，购买/下载所需资源包
  7. 集成 Socket.IO：安装 Socket.IO for Unity（Asset Store 免费/开源）
⚠️ 硬件最低要求（3D开发）：
  • CPU：Intel i5-12代 或 AMD Ryzen 5 5000系列及以上
  • 内存：16GB RAM（推荐32GB）
  • 显卡：NVIDIA GTX 1660 / RTX 2060及以上（6GB+ VRAM）
  • 存储：50GB 可用 SSD 空间
  • 系统：Windows 10/11 64位
