## 一、新角色完整传记

### 伊瑟尔 — 暗影织网者

```
帷幕裂隙的边缘，有一座无人居住的废弃石屋。至少，人们以为它无人居住。

伊瑟尔住在那里面，或者更准确地说——她住在所有屋子的缝隙里。
在帷幕之地，如果你有足够的钱或者足够的秘密，
你迟早会找到伊瑟尔。她什么都知道。

"情报贩子"这个称呼是她自己散布出去的。
更准确的说法是：她是一个在诸神黄昏中失去了自己神祇的祭司。
她侍奉的神——阴影与欺骗之主——在帷幕编织时被用来当作"针线"。
她的神被撕碎、拉长、编织进了帷幕的纤维中。

所以伊瑟尔能听到帷幕中的低语。
不是每个人都能听到的"指引"——而是更深的、更古老的、
属于她那位已经被做成针线的神明的碎片。

她来暮色村不是为了玩游戏。
她来是为了收集足够多的秘密，
用秘密编织成线，用线缝补她神明的碎片，
然后——也许——让祂复活。

而她需要的最后一个秘密，
藏在这场游戏中某个人的血液里。
她不知道是谁。
所以她观察所有人。
```

### 格伦 — 流浪铁匠

```
在矿脉山脉的深处，格伦的家族世世代代为诸神锻造武器。
托尔的雷神之锤，弗雷的神奇宝剑——都出自他们家族之手。

但诸神黄昏后，再没有神来下订单了。

格伦是家族最后一个人。他把铁砧背在背上，
走过整个帷幕之地，为凡人锻造防护装备。
他不收钱——只收"故事"。
因为他说，每一把武器都需要一个故事作为灵魂。

他在暮色村的铁匠铺开张的那天，
一个狼人试图破门而入。
格伦用铁锤敲碎了狼人的下巴，
然后把铁砧堵在门口继续打铁。

第二天早上，所有人都来订做门锁加固。
格伦说："我一晚上只能加固一扇门。选谁，你们自己决定。"
于是每晚，村里人都要争论——
谁最值得被保护。
而格伦只是在旁边静静地打铁，
数着门外的狼嚎声。

他的锤子有一个秘密：
那不是一把普通的铁锤。
那是他家族为托尔锻造雷神之锤时剩下的边角料打的。
虽然不及雷神之锤的万分之一力量，
但在特定的条件下——在被真正需要保护的人手中——
它会发出微弱的雷光。
```

### 艾琳 — 守墓人

```
骨灰平原——帷幕之地最安静的地方。
这里埋着所有在狼人杀循环中死去的人。
每局游戏结束，新的尸体被埋进来。
艾琳负责挖坑、填土、刻名字。

她已经做了二十年。二十年来，她见过太多死亡。
猎人的尸体旁边放着他的猎枪（但第二天枪就消失了——被新的猎人取走了）。
女巫的尸体嘴里塞着最后一片草药叶子。
狼人的尸体在月光下化为灰烬。
村民的尸体——最普通，也最让人难过。

艾琳不说话。但她什么都记得。
谁的尸体是在哪个位置，谁和谁死在同一天晚上，
谁被背叛了（背后有刀伤），谁为了保护别人而死（挡在门前）。

有一天晚上，她发现了一座新的坟墓。
不是她挖的。墓碑上刻着她的名字——艾琳。

她盯着那个墓碑看了一整夜。
天快亮的时候，她拿起铲子，开始挖。
她想知道自己到底死了没有。

然后她挖到了一具空棺。

从此之后，艾琳每晚都到南山墓地来。
不是为了守墓——是为了等那个给她挖坟的人。
她知道，那一定是这场游戏的设计者。
也许是"指引"本身。
也许是帷幕的意志。

她有很多问题想问。
```

### 奥里克 — 疾风信使

```
大瘟疫那年，奥里克十七岁。

瘟疫从一个村子传到另一个村子。
帷幕之地的所有女巫都被隔离了——人们害怕。
没有药，没有医生，没有希望。

奥里克说："我去拿药。"
没有人相信他。
从暮色村到帷幕裂隙，来回六十里。
中间要穿过狼群的领地、沼泽、废墟。
而奥里克只是一个跑得快一点的农家少年。

他用了两天两夜跑完全程。
第三天黎明，他满身泥泞地出现在村口，
手里抱着药瓶——但药瓶是空的。
在路上摔碎了一个，被狼追时丢了一个。
最后只有一个完整的。
只能救一个人。

他选了村里最年轻的女孩——一个五岁的孩子。
其他十三个人在那年冬天死了。
包括他自己的母亲。

事后有人问他："你后悔吗？只救了一个人？"
奥里克说："我后悔没能跑得更快。"

从那以后，他每天都在跑。
跑得更快、更远、更久。
不是为了弥补什么——他自己也说不清。
只是每当他想停下来的时候，
就会听到母亲在那一晚对他说的话：
"去。别管我。去。"

所以现在在暮色村，
当夜幕降临，狼人出没，
奥里克是唯一一个能在村子里自由奔跑的人。
他的速度是他的武器，他的耐力是他的盾牌。
他会把消息送到每一个需要的人手中。
他不会让那一年重演。
```

### 奈莉亚 — 帷幕学者

```
观测塔废墟位于帷幕之地最高的山丘上。
三十年前，奈莉亚来到这里时，塔已经塌了一半。
但另一半——朝向帷幕的那一半——完好无损。

她花了三十年，用唯一的眼睛（另一只在一次"近距离观测"中失明了）
观测帷幕的每一道波纹、每一次闪烁、每一次颜色变化。

她的结论是：帷幕不是静止的。它在呼吸。
它在一吸……一呼……一吸……一呼……
每次呼吸的周期大约是现实中一局游戏的长度。

她发现帷幕最薄弱的时刻——不是血月之夜，不是午夜——
而是当游戏中的"背叛"发生时。
当一个玩家投票出局了自己的同伴。
当狼人刀掉了曾经信任自己的村民。
当守卫没有保护该保护的人，而是在保护自己。

这些背叛——这些"道德破裂"的瞬间——
会在帷幕上激起涟漪。
而这些涟漪，会被帷幕之外的"东西"感知到。

奈莉亚试图警告所有人。但没有人相信她。
所以她自己来了暮色村。
她不信任任何所谓"指引"。
她知道在"指引"背后有某种意志。

她来找出真相。
她来记录一切。
然后——如果必要的话——
她来亲手撕开帷幕，看看外面到底是什么。
```

## 二、世界观深度扩展

### 2.1 帷幕的运作原理

```
帷幕并非一堵"墙"，而是一个"循环"——

1. 诸神用生命编织帷幕，将混沌挡在外面
2. 但帷幕需要"能量"维持
3. 能量来自帷幕之内的人类情感——
   恐惧、希望、背叛、牺牲、信任、猜忌
4. 没有人造这些情感的话，帷幕就会崩溃
5. 所以"指引"出现了——它组织游戏，
   确保总有人在对立，总有恐惧被产生，总有背叛发生
6. 但帷幕不知道的是——这些负面情感的积累，
   也在喂养帷幕之外的存在
7. 这是一个死循环：
   为了生存，必须互相残杀
   互相残杀，让最终威胁越来越近
```

### 2.2 诸神黄昏的真正原因

```
表面上：众神联手对抗帷幕之外的古老存在，全部战死。

真相：众神不是战死的。
他们是自杀的。

当古老存在开始入侵时，众神发现了一个事实：
没有任何武器能伤害它们——
因为它们是由"人类心中的黑暗"构成的。
只要人类还存在恐惧、仇恨、背叛——
这些存在就永远不灭。

解决方案只有一个，但众神花了太久才接受：
他们需要将自己——不朽的、无所不能的神——
转化成一种完全相反的存在：
不是光明，而是"记住光明的黑暗"。
不是守护，而是"通过考验来选择谁值得守护"。
不是爱，而是"让爱在恐惧中被证明"。

所以诸神把自己编织进了帷幕。
他们让自己的存在变成了一道过滤网——
人类在帷幕内的每一次选择，
都是对这些古老存在的投票。
如果人类选择互相保护而非互相残杀，
幕就变厚一分。古老存在就弱一分。
如果人类选择背叛、出卖、自相残杀，
幕就变薄一分。古老存在就强一分。

这就是为什么"指引"组织游戏。
它不是在玩弄人类。
它是在给人类机会——
一次又一次的机会——
去选择正确的道路。

但人类大多数时候选择了什么？
看看你们的投票记录。
```

### 2.3 千面之影

```
它是帷幕之外最古老的存在之一。
它没有固定的形态——它以人类的恐惧为镜子。
你怕什么，它看起来就像什么。

它的目标是进入帷幕之内。
不是为了毁灭——它已经厌倦了混沌的虚无。
它想要体验"存在"——即使以寄生为代价。

它在每一局游戏中挑选一个"容器"——
一个精神最脆弱的玩家。
通过微妙的引导（在指引中插入偏差的低语），
它试图让这个玩家做出最糟糕的选择：
背叛同伴、抛弃责任、自保至上。

如果被选中的容器在游戏结束时存活，
并且做出了"不道德"的选择（由系统根据投票/行动判断），
那么千面之影就在这个容器中埋下了一颗种子。
累积三颗种子的玩家……开始发生变化。
他们的眼睛在夜晚偶尔闪烁。
他们听到更多低语。
他们开始觉得——也许外面的东西没有那么可怕。
也许它们只是孤独。
```

## 三、关键物品扩展描述

### 西格德的猎枪

```
枪托上刻着十二条刻痕——每一道代表一个被这把枪杀死的狼人。
枪管微弯——在近身肉搏中被一只种狼咬弯的。
但那颗子弹还是射出去了。又添了一道刻痕。

西格德从来不擦它。他说血迹是猎枪最好的润滑油。

枪托底部镶着一小片龙骨——
来自他父亲齐格弗里德杀死的那条龙。
所以这把枪在打狼人时，偶尔会发出龙吼般的巨响。
不是因为任何魔法。
只是龙骨的残留意志——对"野兽"的永恒厌恶。
```

### 哈尔瓦德的银链

```
这不是一条项链。这是一个项圈。
哈尔瓦德在十二岁那年自己给自己戴上的。
他知道自己体内有什么。
他知道月圆之夜会发生什么。

银链的内侧刻着一段文字——他母亲在他戴上之前刻下的：
"记住你是一个人。不是因为你没有獠牙，而是因为你能选择不咬人。"

银链在哈尔瓦德变身时依然挂在狼颈上。
它不会断——因为它不是靠物理强度拴住他的。
它靠的是他对那个十二岁小男孩的记忆——
那个还没经历过第一次变身，
还没有在清晨发现手上沾血的男孩。

如果有一天哈尔瓦德彻底放弃了人性，
银链会自动断裂。
目前为止，它还挂着。
```

### 布丽吉德的火把

```
这不是一根普通的火把。
它的燃料不是油、不是木材、不是任何可以耗尽的东西。
它的燃料是——希望。

每次有人在绝境中选择保护他人而非自保，
火把就更亮一点。
每次有人在黑暗中选择说出真相而非沉默，
火焰就更高一点。
每次有人在可以逃跑的时候选择留下，
火就不会熄灭。

相反，每次有人为了自保而出卖同伴，
每次有人在投票中说谎以保护自己，
每次有人放弃抵抗——
火焰就暗一点。

布丽吉德不知道这个机制。
她只知道有时候火焰会毫无原因地变弱——
然后她会在第二天发现某人在投票中背叛了阵营。
她开始怀疑这两者之间的关联。
但还不敢确认。
```

## 四、叙事使用指南

这些故事不是一次性读完的背景材料。
游戏应该**分阶段揭示**——

### 阶段一：选人时
展示角色的标题和一句话简介（代码已有，`getShortDescription()`）

### 阶段二：序幕
展示世界观的简短段落（代码已有，`generatePrologue()`）

### 阶段三：游戏进行中
- 玩家在3D地图中拾取"叙事碎片"时，解锁相关角色的背景段落
- 在特定地标处触发"回声低语"——已死角色的台词语音

### 阶段四：游戏结束后
- 展示本局所有玩家的角色卡（含背景故事）
- 玩家的个人叙事总结（代码已有，`getPersonalNarrativeSummary()`）
- 如果解锁了隐藏关系或特殊成就，展示额外叙事

### 阶段五：长期
- 玩家收集的叙事碎片可在"图鉴"中回顾
- 收集全部碎片解锁世界观"真相文档"
- 累计3局"背叛"行为 → 解锁千面之影相关叙事
- 累计5局"合作"行为 → 解锁诸神真相相关叙事
