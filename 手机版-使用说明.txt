方式一：PWA（手机上直接安装，无需 Android Studio）
──────────────────────────────────────────
1. 先启动服务器：
   - 在电脑上运行政务器 (node server/src/index.js)
   - 或者运行政务器 + 分享你的 WiFi IP 给手机

2. 手机浏览器打开：
   http://你的电脑IP:4000
   例如: http://10.121.197.144:4000

3. 在手机浏览器菜单中选择「添加到主屏幕」
   - Chrome安卓: 右上角 ⋮ → 添加到主屏幕
   - Safari苹果: 底部分享 → 添加到主屏幕

4. 手机桌面上会出现「狼人杀」图标，点击即用！

方式二：打包成 APK 安装包（需要 Android Studio）
──────────────────────────────────────────
1. 安装 Android Studio：
   https://developer.android.com/studio

2. 用 Android Studio 打开此目录：
   client\android\

3. 等待 Gradle 同步完成

4. Build → Build Bundle(s) / APK(s) → Build APK(s)

5. APK 文件生成在：
   client\android\app\build\outputs\apk\debug\app-debug.apk

6. 把 APK 发送到手机安装即可

注意事项：
──────────────────────────────────────────
- 游戏中需要连接服务器，手机和服务器必须能互相访问
- 同一 WiFi 下最简单
- 公网部署可以用 ngrok 等工具
