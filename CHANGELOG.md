## v1.5.1（2026-06-26）🔧 语音修复 + 预言家反馈 + 村民行动

### 修复
- **🎤 语音聊天全面修复**：
  - 修复 STUN URL 拼写错误（缺少 `stun:` 协议前缀导致连接失败）
  - 新增 TURN 服务器中继（openrelay.metered.ca），解决对称 NAT 穿透问题
  - 修复说话指示灯 `isPTT` 闭包过期问题（改用 ref 同步状态）
  - 语音事件节流：只在说话状态变化时发送，减少 60fps → 按需的网络流量
  - ICE 连接状态监控：failed/disconnected 时自动 restartIce，5 秒后仍未恢复则重建连接
- **🛡️ 语音信令安全**：`voice:offer`/`answer`/`iceCandidate` 添加房间成员校验，防止跨房间语音泄漏
- **🔮 预言家查验反馈**：查验后目标头像变为金色（好人）或银白色（狼人），清晰直观
- **⏱️ 夜晚倒计时自动提交**：20 秒倒计时归零时自动提交当前选择（有选择则提交，无则 SLEEP）
- **👨‍🌾 村民夜间行动环节**：村民在所有神和狼之后行动，可出门去别人家并得知访客数量
- **🤖 房主人机数量控制**：房主可通过 +/- 按钮设定本局 AI 数量（0=自动补足）

### 优化改进
- 服务端 `voice:speaking` 添加状态去重，进一步减少冗余广播
- `broadcastNightStepChange()` 增加 `timeLeft` 推送，客户端同步倒计时
- `Player.toPrivateJSON()` 暴露 `checkResult` 和 `checkTarget`
- `Game.getPrivateState()` 新增 `seerCheckResults` 映射表供客户端消费

## v1.5.0（2026-06-26）🤖 AI人机对战

### 新增功能
- **AI人机系统**：新增 BotManager，支持所有 8 种角色的自动决策
  - 房主可在房间设置中开关人机模式
  - 人数不足 6 人时自动补足 AI（可配置）
  - AI 模拟真实玩家行为：随机延迟投票、概率性决策
  - AI 在各夜晚步骤自动行动，支持所有夜晚能力（查验、刀人、守护、毒药、解药等）
  - AI 讨论阶段自动跳过发言
- **断线重连机制**：
  - 玩家断线后 120 秒内可重连，保留角色和状态
  - 超时未重连判定出局，保证游戏平衡
  - 重连时自动迁移所有引用（投票、日志、技能目标、房主身份等）
- **自动更新检测**：客户端启动时比对服务端版本，弹窗提示下载新版本
  - 支持 REST API (`/api/version`) 和 Socket 两种检查方式

### 优化改进
- **夜晚结算器重构**：NightResolver 独立模块，按固定顺序结算
  - 种狼感染预言家特殊处理（预言家保留能力，查验结果反转）
  - 狼人互刀自动相认
  - 守卫重伤机制完善（含毒巫知情）
  - 猎枪腐蚀逻辑优化（先开枪后腐蚀，同晚出门+开枪不冲突）
- **安全增强**：
  - 房间密码 HMAC-SHA256 哈希存储
  - 昵称/聊天消息 XSS 防护（去除 HTML 标签）
  - API 速率限制（创建房间 5次/分钟、加入 10次/分钟、聊天 20次/分钟）
  - 竞态保护：夜晚步骤推进加锁、投票结算去重
- **投票门槛机制**：需要 >50% 存活玩家参与投票，最高票 ≥ 半数才生效，防止单人秒杀
- **版本比较**：客户端自动检测服务端版本，提示更新

### 修复
- 修复房主 hostId 不匹配时自动修正（首位玩家回退逻辑）
- 修复投票结算重复调用导致的竞态问题
- 修复夜晚步骤无存活角色时死锁（自动推进）

## v1.3.0（2026-04）🎯 核心玩法完善

### 新增功能
- **讨论阶段**：投票前轮流发言制，每人 30 秒
  - 按存活玩家顺序自动轮转
  - 玩家可主动跳过自己的发言
  - 房主可手动推进讨论→投票
- **猎人白天开枪**：讨论阶段可开枪击杀一名玩家
- **游戏回放系统**：
  - 每局结束后自动保存回放（参与者、角色、胜负）
  - 每位用户保留最近 5 场回放
  - 可在个人资料中查看历史战绩
- **昼夜日志分离**：`nightLog`（每晚清除）+ `dayLog`（持久保留）
- **房屋访客机制**：
  - 夜晚出门可看到目标屋子的访客数量
  - 村民在 ≥3 人时被赶回家（显示"很多人"）
  - 狼人出门不增加屋子计数

### 优化改进
- 投票结果展示后延迟 2 秒再进入夜晚，给客户端渲染时间
- 夜晚步骤无参与者时随机 3-7 秒延迟后自动推进
- 游戏结束后 30 秒自动返回大厅，5 分钟后清理房间

## v1.0.0（2026-01）🎮 首次发布

### 核心功能
- **完整狼人杀游戏循环**：LOBBY → NIGHT → DAY → VOTE → GAME_OVER
- **8 种角色**：
  | 角色 | 阵营 | 能力 |
  |------|------|------|
  | 🐺 狼人 | 狼人 | 每晚刀人，可去其他狼人家相认 |
  | 👑🐺 种狼 | 狼人 | 可变狼/感染他人；感染预言家保留能力但结果反转 |
  | 🔮 预言家 | 好人 | 每晚查验一名玩家是否为狼人 |
  | ☠️🧪 毒巫 | 好人 | 烈性毒药（毒死一屋子人）+ 药水救人 |
  | 💚🧪 药巫 | 好人 | 万能药（治疗一切含重伤）+ 单目标毒药 |
  | 🛡️ 守卫 | 好人 | 守护目标、重伤机制（≤2人受伤，≥3人死亡） |
  | 🔫 猎人 | 好人 | 第二晚起行动；猎枪追踪射杀 + 短火铳反击 |
  | 👨‍🌾 村民 | 好人 | 夜晚出门观察访客人数 |
- **回合制夜晚**：按固定顺序行动（种狼→守卫→狼人→预言家→毒巫→药巫→结算）
- **房屋位置系统**：每位玩家有独立屋子，夜晚可出门去别人家
- **武器腐蚀机制**：猎枪/短火铳带出门后被腐蚀

### 技术特性
- **Socket.IO 实时通信**：WebSocket 双向通信
- **Express REST API**：`/api/status`、`/api/lobby`、`/api/version`
- **Electron 桌面应用**：Windows 便携版打包（`.exe`）
- **Vite + React 前端**：JSX 组件化架构
- **独立服务器模式**：可部署到云服务器供多人远程联机
- **短房间号**：6 位字母数字组合（排除易混淆字符）
- **角色预设配置**：5-18 人自动分配角色组合

### 客户端组件
- 登录/注册/快速游戏界面
- 大厅房间列表（创建/加入/离开）
- 游戏主面板（玩家列表、阶段指示、夜晚/白天操作）
- 角色卡片展示 + 角色介绍弹窗
- 聊天系统
- 设置面板（音效/显示名称/BGM切换）
